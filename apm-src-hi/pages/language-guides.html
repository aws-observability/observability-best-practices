कस्टम मेट्रिक्स (apm-center सैंपल से)

apm-center में कुछ सैंपल सर्विसेज़ में एक एंडपॉइंट शामिल है जिसका उपयोग परीक्षण के दौरान "कस्टम मेट्रिक्स" व्यवहार का अभ्यास करने के लिए किया जाता है। उदाहरण के लिए, EKS Python सैंपल GET /api/metrics प्रदान करता है:

EKS Python सक्रियण गाइड खोलें

@app.get("/api/metrics")
async def metrics():
    """Custom metrics endpoint"""
    # Simulate some business logic
    processing_time = random.uniform(0.1, 0.5)
    time.sleep(processing_time)

    # Simulate different response times based on some logic
    if processing_time > 0.3:
        status = "slow"
    else:
        status = "fast"

    return {
        "message": "Metrics endpoint",
        "processing_time": processing_time,
        "status": status,
        "timestamp": time.time(),
        "service": "python-eks-service"
    }
