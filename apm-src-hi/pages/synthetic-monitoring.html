सिंथेटिक मॉनिटरिंग (कैनरी)

Synthetics Application Signals से कैसे जुड़ता है

CloudWatch Synthetics कैनरी Application Signals में तब दिखाई दे सकते हैं जब आप कैनरी के लिए X-Ray ट्रेसिंग सक्षम करते हैं। यह कैनरी कॉल को बैकएंड सर्विसेज़ के साथ संबद्ध करने और सर्विस विवरण और Application Map पर दिखाने की अनुमति देता है।

देखें: Application Signals कैसे ऑपरेशनल स्वास्थ्य मॉनिटरिंग सक्षम करता है।

सामान्य कैनरी उपयोग के मामले

  • API हेल्थ चेक: एक शेड्यूल पर प्रमुख एंडपॉइंट को मान्य करें
  • ब्राउज़र फ़्लो: लॉगिन/चेकआउट जर्नी को मान्य करें
  • मल्टी-रीजन: ग्राहकों से पहले रीजनल समस्याओं का पता लगाएं
