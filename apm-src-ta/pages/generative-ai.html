Application Signals + Amazon Bedrock

CloudWatch Application Signals Amazon Bedrock மாடல்களுடன் தொடர்பு கொள்ளும் generative AI பயன்பாடுகளை பிழைத்திருத்துவதை ஆதரிக்கிறது. Bedrock API அழைப்புகளுக்கு out-of-the-box telemetry வழங்குகிறது, இதன்மூலம் நீங்கள் ஆராயலாம்:

  • மாடல் கட்டமைப்பு சிக்கல்கள்
  • பயன்பாடு மற்றும் செலவு காரணிகள் (token பயன்பாடு)
  • மாடல் தாமதம்
  • பதில்களுக்கான நிறுத்தம்/முடிவு காரணங்கள்

நுண்ணிய மெட்ரிக்ஸ் மற்றும் traces

Bedrock அழைப்புகளுக்கு, Application Signals resource-level விவரத்துடன் (எடுத்துக்காட்டாக: model ID, guardrails ID, knowledge base ID, மற்றும் Bedrock agent ID) செயல்திறன் மெட்ரிக்ஸ் மற்றும் trace spans-ஐ உருவாக்கி தொடர்புபடுத்த முடியும்.

பார்க்கவும்: எடுத்துக்காட்டு: Amazon Bedrock-உடன் தொடர்பு கொள்ளும் generative AI apps-ஐ பிழைத்திருத்துதல்.

GenAI semantic attributes + Transaction Search

Application Signals OpenTelemetry GenAI semantic convention attributes-ஐ (எடுத்துக்காட்டாக, gen_ai.request.model மற்றும் token usage fields) வெளியிட முடியும். CloudWatch Transaction Search-உடன் இவற்றை பகுப்பாய்வு செய்து prompts, token பயன்பாடு மற்றும் மாடல்கள் முழுவதும் தாமதத்தை ஒப்பிடலாம்.

  • Transaction Search
  • Spans-ஐ தேடுதல் மற்றும் பகுப்பாய்வு செய்தல்

மேலும் படிக்க

  • AWS அறிவிப்பு: Application Signals Bedrock-ஐ ஆதரிக்கிறது
  • வலைப்பதிவு: AppSignals மூலம் Amazon Bedrock observability-ஐ மேம்படுத்துங்கள்
