தனிப்பயன் metrics (apm-center samples-லிருந்து)

apm-center-ல் உள்ள சில sample சேவைகள் testing-ன் போது "custom metrics" நடத்தையை பயிற்சி செய்ய பயன்படுத்தப்படும் endpoint-ஐ உள்ளடக்குகின்றன. எடுத்துக்காட்டாக, EKS Python sample GET /api/metrics-ஐ வழங்குகிறது:

EKS Python enablement வழிகாட்டியைத் திறக்கவும்

@app.get("/api/metrics")
async def metrics():
    """Custom metrics endpoint"""
    # Simulate some business logic
    processing_time = random.uniform(0.1, 0.5)
    time.sleep(processing_time)

    # Simulate different response times based on some logic
    if processing_time > 0.3:
        status = "slow"
    else:
        status = "fast"

    return {
        "message": "Metrics endpoint",
        "processing_time": processing_time,
        "status": status,
        "timestamp": time.time(),
        "service": "python-eks-service"
    }
