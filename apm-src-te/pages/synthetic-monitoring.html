CloudWatch APM డాక్యుమెంటేషన్

Synthetic Monitoring (కెనరీలు)

Synthetics Application Signals కు ఎలా కనెక్ట్ అవుతుంది

మీరు కెనరీ కోసం X-Ray ట్రేసింగ్‌ను ఎనేబుల్ చేసినప్పుడు CloudWatch Synthetics కెనరీలు Application Signals లో కనిపిస్తాయి. ఇది కెనరీ కాల్‌లను బ్యాకెండ్ సేవలతో అనుబంధించి సేవా వివరాలలో మరియు Application Map పై చూపించడానికి అనుమతిస్తుంది.

చూడండి: Application Signals ఆపరేషనల్ హెల్త్ మానిటరింగ్‌ను ఎలా ఎనేబుల్ చేస్తుంది.

సాధారణ కెనరీ వినియోగ కేసులు

  • API హెల్త్ చెక్‌లు: షెడ్యూల్ ప్రకారం కీ ఎండ్‌పాయింట్‌లను ధృవీకరించండి
  • బ్రౌజర్ ఫ్లోలు: లాగిన్/చెకౌట్ జర్నీలను ధృవీకరించండి
  • బహుళ-ప్రాంతం: కస్టమర్‌ల కంటే ముందు ప్రాంతీయ సమస్యలను గుర్తించండి
