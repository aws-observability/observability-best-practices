发行说明：查看位置

CloudWatch Application Signals 是一项托管的 AWS 功能。公开发行说明和功能公告通过 AWS 文档和 AWS"最新动态"发布。

  • AWS"最新动态"： aws.amazon.com/about-aws/whats-new
  • Application Signals 文档： CloudWatch Application Signals

检测组件

您的"版本化"升级通常来自检测分发包及其依赖项（例如，AWS Distro for OpenTelemetry 和 OpenTelemetry 检测库），以及 CloudWatch Agent / 平台附加组件。

  • ADOT：请按照适合您的语言/运行时的 AWS Distro for OpenTelemetry 发行说明进行操作。
  • CloudWatch Agent：当新的 Application Signals 功能需要时（例如，通过资源属性进行分组），请升级代理。
