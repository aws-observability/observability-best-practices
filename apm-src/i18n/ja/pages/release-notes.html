CloudWatch APM ドキュメント

リリースノート

リリースノートの参照先

CloudWatch Application Signals はマネージド AWS 機能です。公開リリースノートおよび機能に関するお知らせは、AWS ドキュメントおよび AWS「最新情報」を通じて公開されます。

  • AWS「最新情報」: aws.amazon.com/about-aws/whats-new
  • Application Signals ドキュメント: CloudWatch Application Signals

インストルメンテーションコンポーネント

「バージョン管理された」アップグレードは、通常、インストルメンテーションディストリビューションとその依存関係（例：AWS Distro for OpenTelemetry および OpenTelemetry インストルメンテーションライブラリ）、ならびに CloudWatch Agent / プラットフォームアドオンから提供されます。

  • ADOT: ご使用の言語/ランタイムに適した AWS Distro for OpenTelemetry のリリースノートを参照してください。
  • CloudWatch Agent: 新しい Application Signals 機能（例：リソース属性によるグループ化）が必要な場合は、エージェントをアップグレードしてください。
