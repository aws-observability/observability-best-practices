# AWS Config

AWS Config est un service entierement gere qui vous fournit un inventaire des ressources, un historique de configuration et des notifications de changement de configuration pour la securite et la gouvernance. Avec AWS Config, vous pouvez decouvrir les ressources AWS existantes, enregistrer les configurations pour les ressources tierces, exporter un inventaire complet de vos ressources avec tous les details de configuration, et determiner comment une ressource etait configuree a un moment donne. Ces capacites permettent l'audit de conformite, l'analyse de securite, le suivi des changements de ressources et le depannage. AWS Config est concu pour vous aider a superviser vos ressources applicatives dans les scenarios suivants :

1. **Administration des ressources** : AWS Config offre une visibilite sur les configurations des ressources et vous notifie automatiquement des changements de ressources, aidant a detecter les mauvaises configurations et a evaluer la conformite grace aux regles AWS Config.
2. **Audit et conformite** : AWS Config permet de demontrer la conformite en fournissant des donnees de configuration historiques de vos ressources AWS.
3. **Gestion et depannage des changements de configuration** : AWS Config aide a evaluer l'impact des changements de configuration en montrant les relations entre les ressources et fournit des configurations historiques pour le depannage.
4. **Analyse de securite** : AWS Config vous permet d'examiner les permissions IAM historiques et les configurations des groupes de securite pour analyser les vulnerabilites de securite potentielles.
5. **Solutions partenaires** : AWS Config s'integre avec des specialistes tiers de journalisation et d'analyse pour fournir des solutions ameliorees pour la gestion des ressources.
