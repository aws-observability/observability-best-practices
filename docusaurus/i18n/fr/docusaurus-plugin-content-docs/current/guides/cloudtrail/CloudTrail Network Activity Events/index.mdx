# Evenements d'activite reseau CloudTrail

Avec les evenements d'activite reseau pour les points de terminaison VPC, vous pouvez consulter les details de qui accede aux ressources au sein de votre reseau, ce qui vous donne une plus grande capacite a identifier et repondre aux actions malveillantes ou non autorisees dans votre perimetre de donnees. Par exemple, en tant que proprietaire du point de terminaison VPC, vous pouvez consulter les journaux des actions qui ont ete refusees en raison de politiques de point de terminaison VPC ou determiner si un acteur en dehors de votre perimetre de donnees tente d'acceder aux donnees de vos compartiments S3.

### Selecteurs d'evenements avances pour les evenements d'activite reseau
Lorsque vous utilisez des evenements d'activite reseau, les selecteurs d'evenements avances offrent un controle accru sur les evenements CloudTrail ingeres dans vos magasins de donnees d'evenements. Avec les selecteurs d'evenements avances, vous pouvez inclure ou exclure des valeurs sur des champs tels que errorCode, eventName et vpcEndpointId. Les selecteurs d'evenements avances prennent egalement en charge l'inclusion ou l'exclusion de valeurs avec correspondance de motifs sur des chaines partielles. Cela augmente l'efficacite et la precision de vos investigations de securite, de conformite et operationnelles tout en aidant a reduire les couts. Par exemple, vous pouvez filtrer les evenements CloudTrail en fonction de l'attribut vpcEndpointId pour n'inclure que les evenements d'un identifiant de point de terminaison VPC specifique. Cela vous permet de reduire significativement le volume d'evenements CloudTrail ingeres dans CloudTrail Lake, diminuant les couts tout en maintenant la visibilite sur les activites pertinentes des utilisateurs et du systeme.

### Exemples de requetes SQL pour CloudTrail Lake :

#### Nombre d'erreurs d'acces VPC refuse
```
SELECT count(*) as VPCAccessDenied, userIdentity.arn,
    userIdentity.accountid,
    eventName,
    errorMessage,
    vpcEndpointId,
    vpcEndpointAccountId,
    sourceIPAddress
FROM $EDS_ID
WHERE eventCategory = 'NetworkActivity'
    AND eventSource = 's3.amazonaws.com' 
    AND errorCode = 'VpceAccessDenied' 
    AND eventtime >= '2025-02-18 17:00:00'
    AND eventtime <= '2025-02-21 17:00:00'
    GROUP BY 
    userIdentity.arn,
    userIdentity.accountid,
    eventName,
    errorMessage,
    vpcEndpointId,
    vpcEndpointAccountId,
    sourceIPAddress
    ORDER BY eventName DESC;
```

#### Erreurs d'acces VPC refuse
```
SELECT userIdentity.arn,
    userIdentity.accountid,
    eventType,
    eventCategory,
    eventName,
    errorCode,
    errorMessage,
    vpcEndpointId,
    vpcEndpointAccountId,
    sourceIPAddress
FROM $EDS_ID
WHERE eventCategory = 'NetworkActivity'
    AND eventSource = 's3.amazonaws.com' 
    AND errorCode = 'VpceAccessDenied' 
    AND eventtime >= '2025-02-18 17:00:00'
    AND eventtime <= '2025-02-20 17:00:00'
    ORDER BY eventTime DESC;
```
