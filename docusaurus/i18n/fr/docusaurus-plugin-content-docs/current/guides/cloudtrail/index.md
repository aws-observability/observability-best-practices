# AWS CloudTrail

AWS CloudTrail peut être utilisé pour la sécurité, la surveillance des appels API AWS, la notification de dépassements de seuils, les problèmes opérationnels, les mécanismes de filtrage pour isoler les données, l'identification plus rapide des causes profondes et la résolution accélérée. CloudTrail peut également être utilisé pour divers contrôles de conformité et de gouvernance, en vous aidant à atteindre la conformité grâce à la journalisation des appels API et des modifications apportées aux ressources.

Dans la section suivante, nous présenterons certaines des bonnes pratiques pour AWS CloudTrail.
