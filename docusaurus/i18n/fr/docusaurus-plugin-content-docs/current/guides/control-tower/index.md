# AWS Control Tower

AWS Control Tower orchestre d'autres services AWS pour aider les clients à configurer et gouverner rapidement une zone d'atterrissage AWS sécurisée et multi-comptes dans laquelle ils peuvent déployer des charges de travail. Dans cette section, nous passerons en revue les bonnes pratiques que Control Tower met en place en votre nom et suggérerons quelques bonnes pratiques concernant l'exploitation de Control Tower lui-même.
