# AWS Organizations et comptes AWS

Cette section fournit les bonnes pratiques et les conseils operationnels pour la gestion d'AWS Organizations, les strategies multi-comptes et les operations du cycle de vie des comptes, y compris les migrations entre organisations. Les guides couvrent l'evaluation des dependances, la gestion des politiques de gouvernance et les procedures de transfert de comptes en toute securite.
