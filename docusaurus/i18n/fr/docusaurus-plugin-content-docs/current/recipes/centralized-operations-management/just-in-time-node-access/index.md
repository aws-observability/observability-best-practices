# Accès aux noeuds juste-à-temps (JITNA)

Cette section contient une collection d'exemples de ressources pour l'accès aux noeuds juste-à-temps (JITNA) de Systems Manager. Les exemples sont conçus pour aider les clients AWS à comprendre comment implémenter et configurer l'accès aux noeuds juste-à-temps.

Veuillez noter qu'il s'agit de code d'exemple qui doit être testé et validé de manière approfondie dans un environnement de développement avant toute utilisation dans un environnement de production.
