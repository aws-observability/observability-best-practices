# AWS Observability Accelerator

L'AWS Observability Accelerator est un ensemble de modules prescriptifs pour vous aider a configurer l'observabilite de vos environnements AWS avec les services natifs AWS et les services d'observabilite geres par AWS tels qu'Amazon Managed Service for Prometheus, Amazon Managed Grafana, AWS Distro for OpenTelemetry (ADOT) et Amazon CloudWatch.

Nous fournissons une collecte organisee de metriques, journaux et traces, des tableaux de bord CloudWatch, des regles d'alerte et des tableaux de bord Grafana pour votre infrastructure EKS, vos charges de travail basees sur Java/JMX et NGINX, ainsi que vos applications personnalisees.

L'AWS Observability Accelerator fournit des artefacts partages (documentation, tableaux de bord, regles d'alerte) pour les projets [Terraform](https://github.com/aws-observability/terraform-aws-observability-accelerator) et [CDK](https://github.com/aws-observability/cdk-aws-observability-accelerator).

Consultez la documentation du projet pour [Terraform](https://aws-observability.github.io/terraform-aws-observability-accelerator/) et [CDK](https://aws-observability.github.io/cdk-aws-observability-accelerator/) pour plus d'informations.
