# GenAI Observability

AWS पर Generative AI वर्कलोड के लिए Observability -- फाउंडेशन मॉडल, AI एजेंट, प्रॉम्प्ट पाइपलाइन, और मॉडल इनवोकेशन की निगरानी।

## इस अनुभाग में क्या मिलेगा

- **गाइड** -- GenAI Observability के लिए आर्किटेक्चर पैटर्न और डैशबोर्ड
- **रेसिपी** -- Grafana, MCP इंटीग्रेशन, और CloudTrail-आधारित AI वर्कलोड ऑडिटिंग के साथ व्यावहारिक कार्यान्वयन
- **पर्सोना** -- Data Scientist, AI/ML और MLOps इंजीनियरों के लिए सर्वोत्तम प्रथाएँ
