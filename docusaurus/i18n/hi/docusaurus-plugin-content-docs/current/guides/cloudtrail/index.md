# AWS CloudTrail

AWS CloudTrail का उपयोग सुरक्षा, AWS API कॉल्स की निगरानी, थ्रेशोल्ड ब्रीच की अधिसूचना, परिचालन समस्याओं, डेटा को अलग करने के लिए फ़िल्टरिंग मैकेनिज्म, तेज़ रूट कॉज़ पहचान और त्वरित समाधान के लिए किया जा सकता है। CloudTrail का उपयोग विभिन्न अनुपालन और गवर्नेंस नियंत्रणों के लिए भी किया जा सकता है, API कॉल्स और रिसोर्सेज में परिवर्तनों को लॉग करके अनुपालन प्राप्त करने में आपकी मदद करता है।

निम्नलिखित अनुभाग में हम AWS CloudTrail के लिए कुछ बेस्ट प्रैक्टिसेज़ की रूपरेखा प्रस्तुत करेंगे।
