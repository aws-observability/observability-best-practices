# Just-in-time node access (JITNA)

इस अनुभाग में Systems Manager just-in-time node access (JITNA) के लिए नमूना संसाधनों का एक संग्रह है। ये नमूने AWS ग्राहकों को just-in-time node access को लागू और कॉन्फ़िगर करने का तरीका सिखाने के लिए डिज़ाइन किए गए हैं।

कृपया ध्यान रखें कि यह नमूना कोड है और किसी भी प्रोडक्शन वातावरण में उपयोग से पहले डेवलपमेंट वातावरण में अच्छी तरह से परीक्षण और मान्य किया जाना चाहिए।
