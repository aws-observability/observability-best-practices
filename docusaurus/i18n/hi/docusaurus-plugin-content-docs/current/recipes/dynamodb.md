# Amazon DynamoDB

[Amazon DynamoDB][ddb-main] एक की-वैल्यू और डॉक्यूमेंट डेटाबेस है जो किसी भी स्केल पर
सिंगल-डिजिट मिलीसेकंड प्रदर्शन प्रदान करता है। यह एक पूर्णतः प्रबंधित,
मल्टी-रीजन, मल्टी-एक्टिव, ड्यूरेबल डेटाबेस है जिसमें इंटरनेट-स्केल एप्लिकेशन के लिए बिल्ट-इन सुरक्षा, बैकअप और
रिस्टोर, तथा इन-मेमोरी कैशिंग शामिल है।
