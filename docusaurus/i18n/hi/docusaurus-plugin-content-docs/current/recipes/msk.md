# Amazon Managed Streaming for Apache Kafka

[Amazon Managed Streaming for Apache Kafka][msk-main] (MSK) एक पूर्ण रूप से प्रबंधित सेवा है जो Apache Kafka का उपयोग करके स्ट्रीमिंग डेटा को प्रोसेस करने वाले एप्लिकेशन बनाना और चलाना आसान बनाती है। Amazon MSK लगातार क्लस्टर स्वास्थ्य की निगरानी करता है और आपके एप्लिकेशन को बिना किसी डाउनटाइम के अस्वस्थ नोड्स को स्वचालित रूप से बदलता है। इसके अलावा, Amazon MSK डेटा को rest पर एन्क्रिप्ट करके आपके Apache Kafka क्लस्टर को सुरक्षित करता है।
