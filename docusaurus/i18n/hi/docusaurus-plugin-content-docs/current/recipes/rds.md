# Amazon Relational Database Service

[Amazon Relational Database Service][rds-main] (RDS) क्लाउड में रिलेशनल डेटाबेस को सेटअप करना, संचालित करना और स्केल करना आसान बनाता है। यह हार्डवेयर प्रोविजनिंग, डेटाबेस सेटअप, पैचिंग और बैकअप जैसे समय लेने वाले प्रशासनिक कार्यों को स्वचालित करते हुए लागत-कुशल और पुनः आकार देने योग्य क्षमता प्रदान करता है।

निम्नलिखित रेसिपीज देखें:

- [CloudWatch Logs, Lambda, और SNS के साथ RDS के लिए सक्रिय डेटाबेस मॉनिटरिंग बनाएं][rds-cw-sns]
- [CloudWatch का उपयोग करके RDS for PostgreSQL और Aurora for PostgreSQL डेटाबेस लॉग एरर की निगरानी करें और नोटिफिकेशन सेटअप करें][rds-pg-au]
- [Amazon RDS में लॉगिंग और मॉनिटरिंग][rds-mon]
- [CloudWatch में प्रकाशित Performance Insights मेट्रिक्स][rds-pi-cw]
