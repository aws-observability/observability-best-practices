# AWS Cloud Operations इवेंट्स

AWS Cloud Operations के आगामी लाइव इवेंट्स, कार्यशालाओं और एनेबलमेंट सत्रों के बारे में अपडेट रहें।

### आगामी इवेंट्स

AWS Cloud Operations की सर्वोत्तम प्रथाओं, नई सुविधाओं और हैंड्स-ऑन कार्यशालाओं के बारे में जानने के लिए आगामी इवेंट्स के लिए रजिस्टर करें।
