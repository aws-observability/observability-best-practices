# GenAI オブザーバビリティ

AWS 上の生成 AI ワークロードのオブザーバビリティ — 基盤モデル、AI エージェント、プロンプトパイプライン、モデル呼び出しの監視。

## ここで見つけられるもの

- **ガイド** — GenAI オブザーバビリティのアーキテクチャパターンとダッシュボード
- **レシピ** — Grafana、MCP 統合、および CloudTrail ベースの AI ワークロード監査を使用したハンズオン実装
- **ペルソナ** — データサイエンティスト、AI/ML、および MLOps エンジニアのベストプラクティス
