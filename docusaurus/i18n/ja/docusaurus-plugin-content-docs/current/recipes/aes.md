# Amazon OpenSearch Service

[Amazon OpenSearch Service][aes-main] (AOS) は、Amazon Elasticsearch Service の後継サービスであり、インタラクティブなログ分析、リアルタイムアプリケーションモニタリング、ウェブサイト検索などを簡単に実行できます。OpenSearch は、Elasticsearch から派生したオープンソースの分散検索および分析スイートです。OpenSearch の最新バージョン、19 バージョンの Elasticsearch (1.5 から 7.10 バージョン) のサポート、および OpenSearch Dashboards と Kibana (1.5 から 7.10 バージョン) を利用した可視化機能を提供します。
