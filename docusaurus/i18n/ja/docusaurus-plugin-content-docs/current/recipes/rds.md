# Amazon Relational Database Service

[Amazon Relational Database Service][rds-main] (RDS) を使用すると、クラウド内でリレーショナルデータベースを簡単にセットアップ、運用、スケーリングできます。ハードウェアのプロビジョニング、データベースのセットアップ、パッチ適用、バックアップなどの時間のかかる管理タスクを自動化しながら、コスト効率が高く、サイズ変更可能な容量を提供します。

以下のレシピを確認してください。

- [CloudWatch Logs、Lambda、SNS を使用した RDS のプロアクティブなデータベースモニタリングの構築][rds-cw-sns]
- [RDS for PostgreSQL と Aurora for PostgreSQL のデータベースログエラーをモニタリングし、CloudWatch を使用して通知を設定する][rds-pg-au]
- [Amazon RDS でのログ記録とモニタリング][rds-mon]
- [CloudWatch に発行される Performance Insights メトリクス][rds-pi-cw]
