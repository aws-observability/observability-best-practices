# 役立つリンク

AWS オブザーバビリティのベストプラクティスを学び、実装するのに役立つ厳選されたリソースのコレクションです。

### AWS re:Invent 2025 の Cloud Operations (COP) セッション (YouTube プレイリスト)

AWS re:Invent 2025 の Cloud Operations (COP) セッションのブレイクアウトセッションプレイリストは[こちら](https://www.youtube.com/playlist?list=PL2yQDdvlhXf8fUGVeljVNpQwugV7XBZRt)で確認できます。

### インタラクティブデモ

モニタリングとオブザーバビリティ、およびクラウドオペレーション管理のための AWS Cloud Operations デモをご確認ください。

[Storylane で表示](https://aws-cloudops.storylane.io/hub/9ajhvqeu78kb)


### AWS Cloud Operations Show

興味深い AWS Cloud Operations の機能とユースケースを深く掘り下げるライブストリーム録画のコレクションです。このビデオシリーズでは、クラウドオペレーションに関する実践的なデモンストレーションと専門家の洞察を提供します。

[YouTube で視聴する](https://www.youtube.com/playlist?list=PLehXSATXjcQHj8bPSf0uZuQBoxJ7a7ag7)

### One Observability ワークショップ

このハンズオンワークショップでは、AWS が提供する監視とオブザーバビリティのための多様なツールセットについて実践的な経験を得ることができます。このワークショップは高度にカスタマイズ可能で、興味や時間に応じて任意のモジュールを選択できます。

[ワークショップを開始](https://catalog.workshops.aws/observability/)


### Application Signals デモ

AWS Application Signals の機能を紹介する完全なデモンストレーションアプリケーションです。このデモは、実際に動作する例 (Pet Clinic) を提供しており、デプロイして探索することで、Application Signals が実際にどのように機能するかを理解できます。

[GitHub で表示](https://github.com/aws-observability/application-signals-demo)


### AWS オブザーバビリティベストプラクティスガイド

この Observability Best Practices ガイドのホームページです。AWS でのオブザーバビリティの実装に関するベストプラクティス、パターン、レシピを網羅した包括的なリソースです。

[ガイドを参照](/observability-best-practices/ja/)

### AWS クラウド運用ベストプラクティスガイド

Cloud Operations ベストプラクティスガイドのホームページです。この包括的なリソースは、AWS 上で Cloud Operations ワークロードを設定および運用するための推奨アプローチをカバーしています。

[ガイドを参照](https://aws-samples.github.io/cloud-operations-best-practices/)

### AWS Distro for OpenTelemetry (ADOT)

ADOT を使用してコードをインストルメント化する方法に関する包括的なガイドです。AWS がサポートする OpenTelemetry のディストリビューションを使用して、アプリケーションから分散トレースとメトリクスを収集する方法を学びます。

[ADOT ドキュメントを参照](https://aws-otel.github.io/docs/introduction)

### AWS CloudTrail リソース

厳選されたリソースを活用して、**AWS CloudTrail** のセキュリティプラクティスを強化し、インシデント対応を改善し、監視機能を強化します。

[ガイドを参照してください](https://aws-samples.github.io/cloud-operations-best-practices/docs/resources/cloudtrail-resources)
