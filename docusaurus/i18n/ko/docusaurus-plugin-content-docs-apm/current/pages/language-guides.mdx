## 커스텀 메트릭 (apm-center 샘플 기반)

`apm-center`의 일부 샘플 서비스에는 테스트 중 "커스텀 메트릭" 동작을 실행하는 데 사용되는 엔드포인트가 포함되어 있습니다. 예를 들어, EKS Python 샘플은 `GET /api/metrics`를 제공합니다:

[EKS Python 활성화 가이드 열기](/apm/language-guide?combo=eks-python)

```python
@app.get("/api/metrics")
async def metrics():
    """Custom metrics endpoint"""
    # 비즈니스 로직 시뮬레이션
    processing_time = random.uniform(0.1, 0.5)
    time.sleep(processing_time)

    # 로직에 따른 다양한 응답 시간 시뮬레이션
    if processing_time > 0.3:
        status = "slow"
    else:
        status = "fast"

    return {
        "message": "Metrics endpoint",
        "processing_time": processing_time,
        "status": status,
        "timestamp": time.time(),
        "service": "python-eks-service"
    }
```
