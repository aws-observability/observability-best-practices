# 릴리스 노트

## 릴리스 노트: 참조처

CloudWatch Application Signals는 관리형 AWS 기능입니다. 공식 릴리스 노트 및 기능 발표는 AWS 문서와 AWS "What's New"를 통해 게시됩니다.

- **AWS "What's New":** [aws.amazon.com/about-aws/whats-new](https://aws.amazon.com/about-aws/whats-new/)
- **Application Signals 문서:** [CloudWatch Application Signals](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Application-Monitoring-Sections.html)

## 계측 구성 요소

"버전 관리되는" 업그레이드는 일반적으로 계측 배포판과 그 종속성(예: AWS Distro for OpenTelemetry 및 OpenTelemetry 계측 라이브러리), 그리고 CloudWatch Agent / 플랫폼 애드온에서 제공됩니다.

- **ADOT:** 사용하는 언어/런타임에 적합한 AWS Distro for OpenTelemetry 릴리스 노트를 참조하세요.
- **CloudWatch Agent:** 새로운 Application Signals 기능(예: resource attribute를 통한 그룹화)에 필요한 경우 에이전트를 업그레이드하세요.

## 알려진 문제 및 호환성

현재 호환성 매트릭스 및 알려진 문제는 공식 "Supported systems" 페이지를 참조하세요.

[Supported systems](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Application-Signals-supportmatrix.html)
