# Synthetic Monitoring (Canaries)

## Synthetics와 Application Signals의 연동

CloudWatch Synthetics canary에 X-Ray 트레이싱을 활성화하면 Application Signals에 canary가 표시됩니다. 이를 통해 canary 호출을 백엔드 서비스와 연결하고, 서비스 세부 정보 및 **Application Map**에 표시할 수 있습니다.

참고: [How Application Signals enables operational health monitoring](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/Services.html)

## 일반적인 canary 사용 사례

- **API 상태 점검**: 주요 엔드포인트를 일정에 따라 검증
- **브라우저 플로우**: 로그인/결제 여정 검증
- **멀티 리전**: 고객보다 먼저 리전별 문제 감지
