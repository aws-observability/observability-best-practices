# AWS CloudTrail

AWS CloudTrail은 보안, AWS API 호출 모니터링, 임계값 위반 알림, 운영 이슈 파악, 데이터 격리를 위한 필터링 메커니즘, 빠른 근본 원인 식별 및 신속한 해결에 활용할 수 있습니다. 또한 CloudTrail은 API 호출과 리소스 변경 사항을 로깅하여 규정 준수를 달성하는 데 도움을 줌으로써 다양한 컴플라이언스 및 거버넌스 통제에도 사용할 수 있습니다.
