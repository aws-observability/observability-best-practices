# Amazon OpenSearch Service

[Amazon OpenSearch Service][aes-main] (AOS)는 Amazon Elasticsearch Service의 후속 서비스로,
대화형 로그 분석, 실시간 애플리케이션 모니터링, 웹사이트 검색 등을 손쉽게 수행할 수 있도록 해줍니다.
OpenSearch는 Elasticsearch에서 파생된 오픈 소스 분산 검색 및 분석 제품군입니다.
최신 버전의 OpenSearch와 19개 버전의 Elasticsearch(1.5~7.10 버전) 지원,
그리고 OpenSearch Dashboards 및 Kibana(1.5~7.10 버전)를 통한 시각화 기능을 제공합니다.
