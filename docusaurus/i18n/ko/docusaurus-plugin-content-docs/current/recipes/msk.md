# Amazon Managed Streaming for Apache Kafka

[Amazon Managed Streaming for Apache Kafka][msk-main] (MSK)는 Apache Kafka를 사용하여 스트리밍 데이터를 처리하는 애플리케이션을 쉽게 구축하고 실행할 수 있는 완전관리형 서비스입니다. Amazon MSK는 클러스터 상태를 지속적으로 모니터링하고 애플리케이션 다운타임 없이 비정상 노드를 자동으로 교체합니다. 또한 Amazon MSK는 저장 데이터를 암호화하여 Apache Kafka 클러스터를 보호합니다.
