# AWS Observability Accelerator

AWS Observability Accelerator는 Amazon Managed Service for Prometheus, Amazon Managed Grafana, AWS Distro for OpenTelemetry(ADOT) 및 Amazon CloudWatch와 같은 AWS 네이티브 서비스 및 AWS 관리형 Observability 서비스를 사용하여 AWS 환경에 대한 Observability를 설정하는 데 도움이 되는 의견이 반영된 모듈 세트입니다.

EKS 인프라, Java/JMX, NGINX 기반 워크로드 및 사용자 정의 애플리케이션을 위한 큐레이션된 메트릭, 로그, 트레이스 수집, CloudWatch 대시보드, 경보 규칙 및 Grafana 대시보드를 제공합니다.

AWS Observability Accelerator는 [Terraform](https://github.com/aws-observability/terraform-aws-observability-accelerator) 및 [CDK](https://github.com/aws-observability/cdk-aws-observability-accelerator) 프로젝트를 위한 공유 아티팩트(문서, 대시보드, 경보 규칙)를 제공합니다.

자세한 내용은 [Terraform](https://aws-observability.github.io/terraform-aws-observability-accelerator/) 및 [CDK](https://aws-observability.github.io/cdk-aws-observability-accelerator/) 프로젝트 문서를 확인하세요.
