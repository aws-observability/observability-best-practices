# அறிமுகம்

இந்த வழிகாட்டி Application Signals-க்கான தொழில்நுட்ப கட்டமைப்பு, instrumentation உத்திகள் மற்றும் செயல்படுத்தல் அணுகுமுறைகளை ஆராய்கிறது, குறிப்பாக நிறுவனங்கள் ஏன் பாரம்பரிய X-Ray sampling-லிருந்து புதிய விரிவான observability அணுகுமுறைக்கு migrate செய்ய வேண்டும் என்பதில் கவனம் செலுத்துகிறது.
