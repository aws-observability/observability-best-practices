இந்தப் பக்கம் `cw-apm-docs` வழிசெலுத்தலின் ஒரு பகுதியாகும், ஆனால் மூல HTML கோப்பு (`pages/sdk-reference.html`) தற்போதைய `cw-apm-docs` கோப்புறையில் இல்லை.

அதிகாரப்பூர்வ SDK ஆவணங்களைப் பயன்படுத்தவும்:

- AWS SDKs: https://aws.amazon.com/developer/tools/
- CloudWatch API குறிப்பு: https://docs.aws.amazon.com/AmazonCloudWatch/latest/APIReference/Welcome.html
- X-Ray API குறிப்பு: https://docs.aws.amazon.com/xray/latest/api/Welcome.html
