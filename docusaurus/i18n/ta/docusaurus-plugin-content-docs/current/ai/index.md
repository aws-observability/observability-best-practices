# AI

AWS இல் AI-இயக்கப்படும் observability மற்றும் AI பணிச்சுமைகளுக்கான observability.

## [GenAI Observability](./genai/)

Generative AI பணிச்சுமைகளை கண்காணித்தல், தணிக்கை செய்தல் மற்றும் இயக்குதல் — Amazon Bedrock, foundation models, AI agents மற்றும் prompt pipelines.

## [AIOps](./aiops/)

கிளவுட் செயல்பாடுகளை மேம்படுத்த AI மற்றும் machine learning-ஐ பயன்படுத்துதல் — anomaly detection, தானியங்கு root cause analysis, predictive alerting மற்றும் intelligent remediation.
