# AWS Config

AWS Config என்பது வள இருப்பு, கட்டமைப்பு வரலாறு மற்றும் பாதுகாப்பு மற்றும் ஆளுமைக்கான கட்டமைப்பு மாற்ற அறிவிப்புகளை வழங்கும் முழுமையாக நிர்வகிக்கப்படும் சேவையாகும். AWS Config உடன், தற்போதுள்ள AWS வளங்களை கண்டறியலாம், மூன்றாம் தரப்பு வளங்களுக்கான கட்டமைப்புகளை பதிவு செய்யலாம், அனைத்து கட்டமைப்பு விவரங்களுடன் உங்கள் வளங்களின் முழு இருப்பை ஏற்றுமதி செய்யலாம், மற்றும் எந்த நேரத்திலும் ஒரு வளம் எவ்வாறு கட்டமைக்கப்பட்டது என்பதை தீர்மானிக்கலாம். இந்த திறன்கள் இணக்க தணிக்கை, பாதுகாப்பு பகுப்பாய்வு, வள மாற்ற கண்காணிப்பு மற்றும் சரிசெய்தல் ஆகியவற்றைப் பயன்படுத்துகின்றன. AWS Config பின்வரும் சூழ்நிலைகளில் உங்கள் பயன்பாட்டு வளங்களை மேற்பார்வையிட உதவ வடிவமைக்கப்பட்டுள்ளது:

1. **வள நிர்வாகம்**: AWS Config வள கட்டமைப்புகளில் தெரிவுநிலையை வழங்குகிறது மற்றும் வள மாற்றங்களை தானாகவே அறிவிக்கிறது, AWS Config விதிகள் மூலம் தவறான கட்டமைப்புகளை கண்டறிந்து இணக்கத்தை மதிப்பிட உதவுகிறது.
2. **தணிக்கை மற்றும் இணக்கம்**: AWS Config உங்கள் AWS வளங்களின் வரலாற்று கட்டமைப்பு தரவை வழங்குவதன் மூலம் இணக்கத்தை நிரூபிக்க உதவுகிறது.
3. **கட்டமைப்பு மாற்றங்களை நிர்வகித்தல் மற்றும் சரிசெய்தல்**: AWS Config வள உறவுகளைக் காட்டுவதன் மூலம் கட்டமைப்பு மாற்றங்களின் தாக்கத்தை மதிப்பிட உதவுகிறது மற்றும் சரிசெய்தலுக்கான வரலாற்று கட்டமைப்புகளை வழங்குகிறது.
4. **பாதுகாப்பு பகுப்பாய்வு**: AWS Config வரலாற்று IAM அனுமதிகள் மற்றும் பாதுகாப்பு குழு கட்டமைப்புகளை மதிப்பாய்வு செய்து சாத்தியமான பாதுகாப்பு பாதிப்புகளை பகுப்பாய்வு செய்ய அனுமதிக்கிறது.
5. **கூட்டாளர் தீர்வுகள்**: AWS Config மூன்றாம் தரப்பு பதிவு மற்றும் பகுப்பாய்வு நிபுணர்களுடன் ஒருங்கிணைந்து வள மேலாண்மைக்கான மேம்பட்ட தீர்வுகளை வழங்குகிறது.


பின்வரும் பிரிவுகளில் AWS Config க்கான சில சிறந்த நடைமுறைகளை விவரிப்போம்.
