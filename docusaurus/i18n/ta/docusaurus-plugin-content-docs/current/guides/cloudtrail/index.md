# AWS CloudTrail

AWS CloudTrail பாதுகாப்பு, AWS API அழைப்புகளின் கண்காணிப்பு, வரம்பு மீறல்களின் அறிவிப்பு, செயல்பாட்டு சிக்கல்கள், தரவை தனிமைப்படுத்துவதற்கான வடிகட்டல் வழிமுறைகள், விரைவான மூல காரண அடையாளம், மற்றும் விரைவான தீர்வு ஆகியவற்றுக்கு பயன்படுத்தப்படலாம். API அழைப்புகள் மற்றும் வளங்களுக்கான மாற்றங்களை பதிவு செய்வதன் மூலம் இணக்கத்தை அடைய உதவுவதன் மூலம், CloudTrail பல்வேறு இணக்கம் மற்றும் ஆளுமை கட்டுப்பாடுகளுக்கும் பயன்படுத்தப்படலாம்.

பின்வரும் பிரிவில் AWS CloudTrail க்கான சில சிறந்த நடைமுறைகளை விவரிப்போம்.
