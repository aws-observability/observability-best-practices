# டாஷ்போர்டிங்

## சிறந்த நடைமுறைகள்

### உங்கள் முக்கியமான பெர்சோனாக்களுக்கான காட்சிகளை உருவாக்குங்கள்

### உங்கள் வணிக மெட்ரிக்குகளை செயல்பாட்டு மெட்ரிக்குகளுடன் பகிர்ந்து கொள்ளுங்கள்

### டாஷ்போர்டுகள் பாலங்களை உருவாக்க வேண்டும், சுவர்களை அல்ல

### உங்கள் டாஷ்போர்டுகள் ஒரு கதையைச் சொல்ல வேண்டும்

### டாஷ்போர்டுகள் தொழில்நுட்ப மற்றும் தொழில்நுட்பமற்ற பயனர்களுக்கும் கிடைக்க வேண்டும்

## பரிந்துரைகள்

### ஏற்கனவே உள்ள அடையாள வழங்குநர்களைப் பயன்படுத்துங்கள்
