# மெட்ரிக்குகள்

.NET பயன்பாட்டு observability க்கான தரநிலையாக OpenTelemetry ஐ ஏற்றுக்கொண்டுள்ளது, ட்ரேஸ்கள் மற்றும் லாக்குகளுடன் மெட்ரிக்குகள் ஒரு முக்கிய தூணாக உள்ளன. இந்த ஒருங்கிணைப்பு குறைந்தபட்ச மேல்செலவுடன் பயன்பாட்டு செயல்திறனை கண்காணிக்க டெவலப்பர்களுக்கு உதவுகிறது.

.NET சூழலமைப்பில், OpenTelemetry மெட்ரிக்குகள் பயன்பாட்டு மெட்ரிக்குகளை அளவிடுவதற்கும் வெளிப்படுத்துவதற்கும் தரப்படுத்தப்பட்ட அணுகுமுறையை வழங்குகின்றன. .NET 6 இல் தொடங்கி .NET 8 இல் கணிசமாக மேம்படுத்தப்பட்ட இந்த கட்டமைப்பு, மெட்ரிக் தரவை சேகரிப்பதற்கும் ஏற்றுமதி செய்வதற்கும் உள்ளமைந்த ஆதரவை வழங்குகிறது.

ASP.NET Core, HTTP கிளையண்டுகள் மற்றும் Entity Framework போன்ற பொதுவான கூறுகளுக்கு கட்டமைப்பு தானியங்கி இன்ஸ்ட்ருமென்டேஷனை வழங்குகிறது, கூடுதல் குறியீடு இல்லாமல் மதிப்புமிக்க மெட்ரிக்குகளை சேகரிக்கிறது.

.NET இல் OpenTelemetry Prometheus போன்ற பல ஏற்றுமதி வடிவங்களை ஆதரிக்கிறது, இது மெட்ரிக்குகளுக்கு குறிப்பாக பிரபலமானது. இந்த நெகிழ்வுத்தன்மை குழுக்கள் தங்கள் விரும்பிய observability தளங்களுடன் ஒருங்கிணைக்க அனுமதிக்கிறது, அதே நேரத்தில் நிலையான சேகரிப்பு அணுகுமுறையை பராமரிக்கிறது.

OpenTelemetry மெட்ரிக்குகளை ஏற்றுக்கொள்வதன் மூலம், .NET பயன்பாடுகள் மேம்பாட்டு சூழல்களிலிருந்து சிக்கலான உற்பத்தி டிப்ளாய்மென்ட்கள் வரை அளவிடக்கூடிய, விற்பனையாளர்-நடுநிலை, தரப்படுத்தப்பட்ட கண்காணிப்பு அணுகுமுறையிலிருந்து பயனடைகின்றன, பயன்பாட்டு ஆரோக்கியம் மற்றும் செயல்திறன் பற்றிய முக்கியமான தெரிவுநிலையை வழங்குகின்றன.

## மெட்ரிக்குகள் செயல்படுத்தல்

.NET 8 பயன்பாடுகளில் OpenTelemetry மெட்ரிக்குகளை செயல்படுத்துவது மிகவும் எளிமையாகிவிட்டது. கட்டமைப்பு செயல்முறை நவீன .NET பயன்பாடுகளுக்கு மையமான dependency injection அமைப்பை பயன்படுத்துகிறது. பயன்பாட்டு bootstrap செயல்முறையின் போது fluent API ஐப் பயன்படுத்தி மெட்ரிக்குகள் சேகரிப்பை டெவலப்பர்கள் கட்டமைக்கலாம், இது நோக்கத்தை தெளிவாகவும் கட்டமைப்பு விருப்பங்களை கண்டுபிடிக்கக்கூடியதாகவும் ஆக்குகிறது:

```c#
var builder = WebApplication.CreateBuilder(args);

// Add OpenTelemetry metrics
builder.Services.AddOpenTelemetry()
    .WithMetrics(metrics => metrics
        .AddAspNetCoreInstrumentation()
        .AddHttpClientInstrumentation()
        .AddRuntimeInstrumentation()
        .AddOtlpExporter());
```

## தனிப்பயன் மெட்ரிக்குகள்

System.Diagnostics.Metrics நேம்ஸ்பேஸைப் பயன்படுத்தி டெவலப்பர்கள் தனிப்பயன் மெட்ரிக்குகளை உருவாக்கலாம்:

```c#
using var meter = new Meter("MyApplication.Metrics");
var orderCounter = meter.CreateCounter<int>("orders.processed");

// Recording values
orderCounter.Add(1, new("customer", customerId));
```
