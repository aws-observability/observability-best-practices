# AWS பகுதி பயன்பாடு

AWS பகுதிகள் உங்கள் கிளவுட் கட்டிடக்கலையின் அடிப்படை கட்டுமான தொகுதிகளாகும், புவியியல் ரீதியாக தனிமைப்படுத்தப்பட்ட இடங்களாக செயல்படுகின்றன, அவை பல Availability Zones களை கொண்டுள்ளன. செயல்திறனை மேம்படுத்துதல், தரவு வதிவிட தேவைகளுக்கான இணக்கத்தை உறுதிசெய்தல், செலவுகளை நிர்வகித்தல் மற்றும் நெகிழ்வான பேரிடர் மீட்பு உத்திகளை உருவாக்குதல் ஆகியவற்றுக்கு சரியான பகுதிகளை தேர்வு செய்வது மிக முக்கியமானது. ஒவ்வொரு பகுதியும் சுயாதீனமாக செயல்படுகிறது, இயற்கையான பிழை தனிமைப்படுத்தலை வழங்குகிறது மற்றும் தரவு இறையாண்மையை பராமரிக்கும் அதே நேரத்தில் உலகளாவிய விநியோகிக்கப்பட்ட பயன்பாடுகளை வடிவமைக்க உங்களை இயலச் செய்கிறது. இந்த பிரிவில், பகுதி இயக்குதல் உத்திகள், தேர்வு அளவுகோல்கள் மற்றும் உங்கள் செயல்பாட்டு மற்றும் வணிக தேவைகளை பூர்த்தி செய்ய AWS பகுதிகளை திறம்பட பயன்படுத்துவதற்கான சிறந்த நடைமுறைகள் குறித்த வழிகாட்டுதலை வழங்குவோம்.
