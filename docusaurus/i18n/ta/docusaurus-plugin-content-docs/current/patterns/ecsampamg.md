# ECS பணிச்சுமைகளைக் கண்காணித்தல்
<!--with ADOT, AWS X-Ray, and Amazon Managed Service for Prometheus-->

## அறிமுகம்

கண்டெய்னரைஸ்டு அப்ளிகேஷன்களின் உலகில், நம்பகத்தன்மை மற்றும் செயல்திறனைப் பராமரிக்க திறம்பட கண்காணிப்பது மிக முக்கியமானது. இந்த ஆவணம் Amazon Elastic Container Service (ECS) பணிச்சுமைகளுக்கான மேம்பட்ட கண்காணிப்பு தீர்வை விவரிக்கிறது, AWS Distro for OpenTelemetry (ADOT), AWS X-Ray மற்றும் Amazon Managed Service for Prometheus ஆகியவற்றைப் பயன்படுத்துகிறது.

## கட்டமைப்பு கண்ணோட்டம்

கண்காணிப்பு கட்டமைப்பு அப்ளிகேஷன் மற்றும் ADOT கலெக்டர் இரண்டையும் ஹோஸ்ட் செய்யும் ECS டாஸ்க்கை மையமாகக் கொண்டுள்ளது. இந்த அமைவு அப்ளிகேஷன் சூழலிலிருந்து நேரடியாக விரிவான தரவு சேகரிப்பை இயக்குகிறது.

## முக்கிய கூறுகள்

### ECS டாஸ்க்
ECS டாஸ்க் அடிப்படை அலகாகச் செயல்படுகிறது, அப்ளிகேஷன் மற்றும் கண்காணிப்பு கூறுகளை உள்ளடக்குகிறது.

### மாதிரி அப்ளிகேஷன்
ECS டாஸ்க்கிற்குள் இயங்கும் கண்டெய்னரைஸ்டு அப்ளிகேஷன், கண்காணிக்கப்பட வேண்டிய பணிச்சுமையைப் பிரதிநிதித்துவப்படுத்துகிறது.

### AWS Distro for OpenTelemetry (ADOT) Collector
அப்ளிகேஷனுடன் சேர்ந்து டிப்ளாய் செய்யப்படும் ADOT கலெக்டர், டெலிமெட்ரி தரவுக்கான மைய ஒருங்கிணைப்பு புள்ளியாகச் செயல்படுகிறது. இது அப்ளிகேஷனிலிருந்து மெட்ரிக்குகள் மற்றும் ட்ரேஸ்கள் இரண்டையும் சேகரிக்கிறது.

### AWS X-Ray
X-Ray ADOT கலெக்டரிடமிருந்து ட்ரேஸ் தரவைப் பெறுகிறது, கோரிக்கை ஓட்டங்கள் மற்றும் சேவை சார்புநிலைகள் பற்றிய விரிவான நுண்ணறிவுகளை வழங்குகிறது.

### Amazon Managed Service for Prometheus
இந்த சேவை ADOT கலெக்டரால் சேகரிக்கப்பட்ட மெட்ரிக்குகளைச் சேமித்து நிர்வகிக்கிறது, மெட்ரிக் சேமிப்பு மற்றும் வினவலுக்கான அளவிடக்கூடிய தீர்வை வழங்குகிறது.

## தரவு ஓட்டம்

1. மாதிரி அப்ளிகேஷன் அதன் செயல்பாட்டின் போது டெலிமெட்ரி தரவை உருவாக்குகிறது.
2. அதே ECS டாஸ்க்கில் இயங்கும் ADOT கலெக்டர் அப்ளிகேஷனிலிருந்து இந்தத் தரவைச் சேகரிக்கிறது.
3. ட்ரேஸ் தரவு விநியோகிக்கப்பட்ட ட்ரேசிங் பகுப்பாய்வுக்காக AWS X-Ray-க்கு அனுப்பப்படுகிறது.
4. மெட்ரிக்குகள் சேமிப்பு மற்றும் பிற்கால பகுப்பாய்வுக்காக Amazon Managed Service for Prometheus-க்கு அனுப்பப்படுகின்றன.

## நன்மைகள்

- **விரிவான கண்காணிப்பு**: மெட்ரிக்குகள் மற்றும் ட்ரேஸ்கள் இரண்டையும் கைப்பற்றி, அப்ளிகேஷன் செயல்திறனின் முழுமையான காட்சியை வழங்குகிறது.
- **அளவிடக்கூடியது**: பெரிய அளவிலான டெலிமெட்ரி தரவைக் கையாள நிர்வகிக்கப்படும் சேவைகளைப் பயன்படுத்துகிறது.
- **ஒருங்கிணைப்பு**: ECS மற்றும் பிற AWS சேவைகளுடன் தடையின்றி வேலை செய்கிறது.
- **குறைக்கப்பட்ட செயல்பாட்டு மேல்நிலை**: உள்கட்டமைப்பு மேலாண்மையின் தேவையைக் குறைக்க நிர்வகிக்கப்படும் சேவைகளைப் பயன்படுத்துகிறது.

## செயல்படுத்தல் கருத்தில் கொள்ள வேண்டியவை

- X-Ray மற்றும் Prometheus-க்கு தரவு பரிமாற்றத்தை அனுமதிக்க ECS டாஸ்க்கிற்கு சரியான IAM பாத்திரங்கள் மற்றும் அனுமதிகள் உள்ளமைக்கப்பட வேண்டும்.
- ECS டாஸ்க்கிற்குள் வள ஒதுக்கீடு அப்ளிகேஷன் மற்றும் ADOT கலெக்டர் இரண்டையும் கணக்கில் எடுத்துக்கொள்ள வேண்டும்.
- முழுமையான Observability தீர்வுக்கு மெட்ரிக்குகள் மற்றும் ட்ரேஸ்களுடன் சேர்த்து லாக் சேகரிப்பையும் செயல்படுத்துவதைக் கருத்தில் கொள்ளுங்கள்.

## முடிவு

இந்த கட்டமைப்பு ECS பணிச்சுமைகளுக்கான வலுவான கண்காணிப்பு தீர்வை வழங்குகிறது, OpenTelemetry-ன் சக்தியை AWS நிர்வகிக்கப்படும் சேவைகளுடன் இணைக்கிறது. இது அப்ளிகேஷன் செயல்திறன் மற்றும் நடத்தையில் ஆழமான நுண்ணறிவுகளை இயக்குகிறது, கண்டெய்னரைஸ்டு சூழல்களுக்கான விரைவான சிக்கல் தீர்வு மற்றும் தகவலறிந்த முடிவெடுத்தலை எளிதாக்குகிறது.
