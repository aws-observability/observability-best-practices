# Just-in-time node access (JITNA)

இந்த பிரிவு Systems Manager just-in-time node access (JITNA) க்கான மாதிரி ஆதாரங்களின் தொகுப்பை உள்ளடக்கியது. AWS வாடிக்கையாளர்களுக்கு just-in-time node access ஐ எவ்வாறு செயல்படுத்துவது மற்றும் கட்டமைப்பது என்பதை கற்பிக்க இந்த மாதிரிகள் வடிவமைக்கப்பட்டுள்ளன.

இது மாதிரி குறியீடு என்பதையும், உற்பத்தி சூழலில் பயன்படுத்துவதற்கு முன் மேம்பாட்டு சூழலில் முழுமையாக சோதிக்கப்பட்டு சரிபார்க்கப்பட வேண்டும் என்பதையும் நினைவில் கொள்ளுங்கள்.
