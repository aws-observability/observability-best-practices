# பாதுகாப்பு சம்பவ மறுமொழி மற்றும் ஃபொரன்சிக் பகுப்பாய்வு

AWS CloudTrail உங்கள் AWS உள்கட்டமைப்பு முழுவதும் கணக்கு செயல்பாட்டைக் கண்காணிக்கவும் பதிவு செய்யவும் ஒரு முக்கியமான சேவையாகும். இது API அழைப்புகள், பயனர் செயல்கள் மற்றும் சேவை நிகழ்வுகளின் விரிவான பதிவுகளை வழங்குகிறது, இவை பாதுகாப்பு ஃபொரன்சிக்ஸ், சம்பவ மறுமொழி மற்றும் இணக்கத்தன்மை தணிக்கைக்கு அவசியமானவை.

இந்த பிரிவு பாதுகாப்பு சம்பவ மறுமொழி மற்றும் ஃபொரன்சிக் பகுப்பாய்வுக்கு CloudTrail-ஐ பயன்படுத்துவது குறித்த விரிவான வழிகாட்டுதலை வழங்குகிறது. முக்கியமான நிகழ்வு புலங்களைப் புரிந்துகொள்வது, பாதுகாப்பு கண்காணிப்புக்கான சிறந்த நடைமுறைகளை செயல்படுத்துவது மற்றும் உங்கள் பாதுகாப்பு நிலையை வலுப்படுத்த தொகுக்கப்பட்ட ரிசோர்ஸ்களை அணுகுவது ஆகியவற்றை நீங்கள் கற்றுக்கொள்வீர்கள்.

### நீங்கள் கற்றுக்கொள்வது

**[CloudTrail நிகழ்வு புலங்களைப் புரிந்துகொள்ளுதல்](./event-fields.mdx)**
- முக்கியமான CloudTrail நிகழ்வு புலங்கள் மற்றும் அவற்றின் பாதுகாப்பு முக்கியத்துவம்
- சமரசமான அடையாளங்களை அடையாளம் காணவும், தாக்குபவர் செயல்களைக் கண்காணிக்கவும், குறுக்கு-கணக்கு அணுகலைக் கண்டறியவும் எவ்வாறு
- ஃபொரன்சிக் பகுப்பாய்வு மற்றும் சம்பவ மறுமொழிக்கான நடைமுறை பயன்பாட்டு நிகழ்வுகள்
- சமரசமான AWS access keys-ஐ விசாரிக்கும் உதாரண பகுப்பாய்வு பணிப்பாய்வு

**[பாதுகாப்பு ஃபொரன்சிக்ஸ் சிறந்த நடைமுறைகள்](./best-practice-security-forensics.mdx)**
- அனைத்து பகுதிகள் மற்றும் கணக்குகளில் விரிவான லாக்கிங் கட்டமைப்பு
- CloudTrail Lake மற்றும் Amazon Athena பயன்படுத்தி மேம்பட்ட வினவல் நுட்பங்கள்
- சந்தேகத்திற்குரிய செயல்பாடுகளுக்கான நிகழ்நேர கண்காணிப்பு மற்றும் எச்சரிக்கை
- லாக் பாதுகாப்பு மற்றும் ஒருமைப்பாடு பாதுகாப்பு நடவடிக்கைகள்
- CloudTrail Insights-உடன் அசாதாரண கண்டறிதல்

**[கூடுதல் பாதுகாப்பு ரிசோர்ஸ்கள்](../../../resources/cloudtrail-resources)**
- சம்பவ விசாரணை, கண்காணிப்பு மற்றும் தானியக்கத்திற்கான தொகுக்கப்பட்ட AWS வலைப்பதிவு இடுகைகள்
- நிஜ-உலக தாக்குதல் காட்சிகளை உருவகப்படுத்தும் நடைமுறை AWS CloudTrail பாதுகாப்பு பட்டறைகள்
- CloudTrail பகுப்பாய்வு மற்றும் சம்பவ மறுமொழியை தானியக்கமாக்கும் கருவிகள் மற்றும் ஸ்கிரிப்ட்கள்

### முக்கிய நன்மைகள்

- **மேம்பட்ட பாதுகாப்பு நிலை**: CloudTrail-ன் விரிவான லாக்கிங் திறன்களைப் பயன்படுத்தி பாதுகாப்பு சம்பவங்களைக் கண்டறிந்து பதிலளிக்க கற்றுக்கொள்ளுங்கள்
- **ஃபொரன்சிக் பகுப்பாய்வு**: முக்கியமான நிகழ்வு புலங்கள் மற்றும் அவற்றின் உறவுகளைப் புரிந்துகொள்வதன் மூலம் பாதுகாப்பு நிகழ்வுகளை விசாரிக்கும் கலையை கற்றுத்தேர்ந்துகொள்ளுங்கள்
- **இணக்கத்தன்மை ஆதரவு**: சரியான தணிக்கை தடங்கள் மற்றும் கண்காணிப்பு நடைமுறைகளுடன் ஒழுங்குமுறை தேவைகளை பூர்த்தி செய்யுங்கள்
- **தானியங்கு மறுமொழி**: நிகழ்நேர எச்சரிக்கை மற்றும் தானியங்கு சம்பவ மறுமொழி பணிப்பாய்வுகளை செயல்படுத்துங்கள்
- **செலவு உகப்பாக்கம்**: advanced event selectors மூலம் லாக்கிங் செலவுகளைக் கட்டுப்படுத்தும் அதே நேரத்தில் உயர் அபாய நிகழ்வுகளில் கவனம் செலுத்துங்கள்

### தொடங்குதல்

பயனுள்ள பாதுகாப்பு பகுப்பாய்வுக்கான அடித்தளத்தை உருவாக்க [முக்கியமான CloudTrail நிகழ்வு புலங்களைப்](./event-fields.mdx) புரிந்துகொள்வதில் தொடங்குங்கள். பின்னர் உங்கள் சூழலில் விரிவான பாதுகாப்பு கண்காணிப்பை செயல்படுத்த [சிறந்த நடைமுறைகளை](./best-practice-security-forensics.mdx) ஆராயுங்கள்.

நடைமுறை அனுபவத்திற்கு, உருவகப்படுத்தப்பட்ட பாதுகாப்பு சம்பவங்களைக் கண்டறிந்து பதிலளிப்பதை பயிற்சி செய்ய எங்கள் [கூடுதல் ரிசோர்ஸ்கள்](../../../resources/cloudtrail-resources) பிரிவில் குறிப்பிடப்பட்டுள்ள AWS பட்டறைகளை முயற்சிக்கவும்.
