# Amazon DynamoDB

[Amazon DynamoDB][ddb-main] என்பது எந்த அளவிலும் ஒற்றை-இலக்க மில்லிவினாடி செயல்திறனை வழங்கும் ஒரு key-value மற்றும் document தரவுத்தளமாகும். இது முழுமையாக நிர்வகிக்கப்படும், பல-பிராந்திய, பல-செயலில் உள்ள, நீடித்த தரவுத்தளமாகும், இதில் உள்ளமைக்கப்பட்ட பாதுகாப்பு, காப்புப்பிரதி மற்றும் மீட்டெடுப்பு, மற்றும் இணைய-அளவிலான பயன்பாடுகளுக்கான நினைவக கேச்சிங் ஆகியவை உள்ளன.
