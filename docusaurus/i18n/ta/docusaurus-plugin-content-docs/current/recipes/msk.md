# Amazon Managed Streaming for Apache Kafka

[Amazon Managed Streaming for Apache Kafka][msk-main] (MSK) என்பது ஸ்ட்ரீமிங் தரவை செயலாக்க Apache Kafka பயன்படுத்தும் பயன்பாடுகளை உருவாக்கி இயக்குவதை உங்களுக்கு எளிதாக்கும் முழுமையாக நிர்வகிக்கப்படும் சேவையாகும். Amazon MSK உங்கள் பயன்பாட்டிற்கு எந்த வேலையில்லா நேரமும் இல்லாமல் கிளஸ்டர் ஆரோக்கியத்தை தொடர்ச்சியாக கண்காணித்து தானாகவே ஆரோக்கியமற்ற நோடுகளை மாற்றுகிறது. கூடுதலாக, Amazon MSK ஓய்வு நிலையில் தரவை மறையாக்கம் செய்வதன் மூலம் உங்கள் Apache Kafka கிளஸ்டரை பாதுகாக்கிறது.
