# Amazon Relational Database Service

[Amazon Relational Database Service][rds-main] (RDS) கிளவுட்டில் ஒரு தொடர்புடைய தரவுத்தளத்தை அமைப்பது, இயக்குவது மற்றும் அளவிடுவதை எளிதாக்குகிறது. வன்பொருள் வழங்கல், தரவுத்தள அமைப்பு, இணைப்பு மற்றும் காப்புப்பிரதிகள் போன்ற நேரத்தை எடுக்கும் நிர்வாகப் பணிகளை தானியங்குபடுத்தும் அதே வேளையில் செலவு-திறனான மற்றும் அளவை மாற்றக்கூடிய திறனை வழங்குகிறது.

பின்வரும் செய்முறைகளைப் பாருங்கள்:

- [CloudWatch Logs, Lambda மற்றும் SNS மூலம் RDS-க்கான முன்கூட்டிய தரவுத்தள கண்காணிப்பை உருவாக்குதல்][rds-cw-sns]
- [PostgreSQL-க்கான RDS மற்றும் PostgreSQL-க்கான Aurora தரவுத்தள லாக் பிழைகளை கண்காணித்து CloudWatch பயன்படுத்தி அறிவிப்புகளை அமைத்தல்][rds-pg-au]
- [Amazon RDS-இல் லாக்கிங் மற்றும் கண்காணிப்பு][rds-mon]
- [CloudWatch-க்கு வெளியிடப்பட்ட Performance Insights மெட்ரிக்குகள்][rds-pi-cw]
