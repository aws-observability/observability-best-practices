## AWS CloudTrail Workshops

இந்த hands-on workshops பொதுவான security events-ஐ simulate செய்து AWS CloudTrail மற்றும் தொடர்புடைய services பயன்படுத்தி detection மற்றும் response-ஐ demonstrate செய்கின்றன. ஒவ்வொரு workshop-லும் real-world scenarios-ஐ replicate செய்ய CloudFormation template அல்லது script உள்ளது.

### Workshop பட்டியல்
- **[அங்கீகரிக்கப்படாத IAM Credential பயன்பாடு](https://catalog.us-east-1.prod.workshops.aws/workshops/6a8ad836-10a6-4694-9a3b-f53f193041de/en-US/setup/cloudformation)**  
  **விளக்கம்**: CloudShell script வழியாக அங்கீகரிக்கப்படாத IAM credential பயன்பாட்டை simulate செய்தல், reconnaissance மற்றும் privilege escalation-ஐ mimicking செய்தல்.  
  **ஏன் முக்கியம்**: AWS environments-ல் பொதுவான attack vector ஆன அங்கீகரிக்கப்படாத அணுகலை detect செய்யவும் mitigate செய்யவும் கற்றுக்கொள்ளுங்கள்.

- **[S3-ல் Ransomware](https://catalog.workshops.aws/aws-cirt-ransomware-simulation-and-detection/en-US)**  
  **விளக்கம்**: S3 buckets உருவாக்கி CloudShell-ல் data exfiltration மற்றும் deletion வழியாக ransomware-ஐ simulate செய்ய CloudFormation template பயன்படுத்துதல்.  
  **ஏன் முக்கியம்**: அங்கீகரிக்கப்படாத S3 modifications-ஐ detect செய்வதன் மூலம் data loss அல்லது exposure-லிருந்து பாதுகாத்தல்.

- **[Cryptominer Security Events](https://catalog.workshops.aws/aws-cirt-cryptominer-simulation-and-detection/en-US)**  
  **விளக்கம்**: அறியப்பட்ட domains-க்கு DNS requests மூலம் cryptomining-ஐ simulate செய்ய CloudFormation வழியாக EC2 instances deploy செய்தல்.  
  **ஏன் முக்கியம்**: செலவுகளை குறைக்கவும் EC2 instances-ஐ பாதுகாக்கவும் resource-intensive cryptomining-ஐ கண்டறிதல்.

- **[IMDSv1-ல் SSRF](https://catalog.us-east-1.prod.workshops.aws/workshops/6a8ad836-10a6-4694-9a3b-f53f193041de/en-US/setup/ssrf)**  
  **விளக்கம்**: IMDSv1 vulnerabilities வழியாக EC2 instance credentials-ல் server-side request forgery (SSRF) attack-ஐ simulate செய்தல்.  
  **ஏன் முக்கியம்**: SSRF mitigation மூலம் instance credentials-க்கான அங்கீகரிக்கப்படாத அணுகலை தடுத்தல்.
