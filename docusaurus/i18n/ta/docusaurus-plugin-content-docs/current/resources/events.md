# AWS Cloud Operations நிகழ்வுகள்

AWS Cloud Operations-க்கான வரவிருக்கும் live events, workshops மற்றும் enablement sessions பற்றி புதுப்பித்த நிலையில் இருங்கள்.

### வரவிருக்கும் நிகழ்வுகள்

AWS Cloud Operations சிறந்த நடைமுறைகள், புதிய அம்சங்கள் மற்றும் hands-on workshops பற்றி கற்றுக்கொள்ள வரவிருக்கும் நிகழ்வுகளுக்கு பதிவு செய்யுங்கள்.
