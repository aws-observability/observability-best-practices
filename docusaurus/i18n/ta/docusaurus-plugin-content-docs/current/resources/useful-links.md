### AWS Distro for OpenTelemetry (ADOT)

ADOT-உடன் உங்கள் code-ஐ instrument செய்வது பற்றிய விரிவான guides. AWS-supported OpenTelemetry distribution பயன்படுத்தி உங்கள் applications-லிருந்து distributed traces மற்றும் metrics-ஐ எவ்வாறு சேகரிப்பது என்பதை கற்றுக்கொள்ளுங்கள்.

[ADOT ஆவணத்தை ஆராய](https://aws-otel.github.io/docs/introduction)

### AWS CloudTrail வளங்கள்

உங்கள் **AWS CloudTrail** பாதுகாப்பு நடைமுறைகளை வலுப்படுத்தவும், incident response-ஐ மேம்படுத்தவும், monitoring திறன்களை மேம்படுத்தவும் தொகுக்கப்பட்ட வளங்களை ஆராயுங்கள்.
