# AWS Observability Accelerator

AWS Observability Accelerator என்பது Amazon Managed Service for Prometheus, Amazon Managed Grafana, AWS Distro for OpenTelemetry (ADOT) மற்றும் Amazon CloudWatch போன்ற AWS நேட்டிவ் சேவைகள் மற்றும் AWS நிர்வகிக்கப்பட்ட Observability சேவைகளைப் பயன்படுத்தி உங்கள் AWS சூழல்களுக்கான Observability-ஐ அமைக்க உதவும் கருத்துடைய தொகுதிகளின் தொகுப்பாகும்.

உங்கள் EKS உள்கட்டமைப்பு, Java/JMX, NGINX அடிப்படையிலான பணிச்சுமைகள் மற்றும் உங்கள் தனிப்பயன் பயன்பாடுகளுக்கான தேர்ந்தெடுக்கப்பட்ட மெட்ரிக்குகள், லாக்குகள், ட்ரேஸ்கள் சேகரிப்பு, CloudWatch டாஷ்போர்டு, எச்சரிக்கை விதிகள் மற்றும் Grafana டாஷ்போர்டுகளை வழங்குகிறோம்.

AWS Observability Accelerator [Terraform](https://github.com/aws-observability/terraform-aws-observability-accelerator) மற்றும் [CDK](https://github.com/aws-observability/cdk-aws-observability-accelerator) திட்டங்களுக்கான பகிரப்பட்ட கலைப்பொருட்களை (ஆவணங்கள், டாஷ்போர்டுகள், எச்சரிக்கை விதிகள்) வழங்குகிறது.

மேலும் தகவலுக்கு [Terraform](https://aws-observability.github.io/terraform-aws-observability-accelerator/) மற்றும் [CDK](https://aws-observability.github.io/cdk-aws-observability-accelerator/) திட்ட ஆவணங்களைப் பாருங்கள்.
