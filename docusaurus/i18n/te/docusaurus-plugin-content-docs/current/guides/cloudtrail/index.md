# AWS CloudTrail

AWS CloudTrail security, AWS API calls monitoring, threshold breaches notification, operational issues, data isolate చేయడానికి filtering mechanisms, faster root cause identification మరియు speedy resolution కోసం ఉపయోగించవచ్చు. API calls మరియు resources కు changes ను log చేయడం ద్వారా compliance మరియు governance controls ను సాధించడంలో CloudTrail మీకు సహాయపడుతుంది.

కింది విభాగంలో AWS CloudTrail కోసం కొన్ని best practices ను outline చేస్తాము.
