# Just-in-time node access (JITNA)

ఈ విభాగం Systems Manager just-in-time node access (JITNA) కోసం నమూనా resources సేకరణను కలిగి ఉంది. ఈ నమూనాలు AWS కస్టమర్లకు just-in-time node access ని ఎలా implement చేయాలి మరియు configure చేయాలి అనే విషయంపై అవగాహన కల్పించడానికి రూపొందించబడ్డాయి.

దయచేసి గుర్తుంచుకోండి, ఇది నమూనా code మాత్రమే మరియు ఏదైనా production environment లో ఉపయోగించడానికి ముందు development environment లో పూర్తిగా పరీక్షించి ధృవీకరించాలి.
