# Amazon Managed Streaming for Apache Kafka

[Amazon Managed Streaming for Apache Kafka][msk-main] (MSK) అనేది స్ట్రీమింగ్ డేటాను ప్రాసెస్ చేయడానికి Apache Kafka ను ఉపయోగించే అప్లికేషన్‌లను నిర్మించడం మరియు నడపడం సులభం చేసే పూర్తిగా నిర్వహించబడే సర్వీస్. Amazon MSK క్లస్టర్ ఆరోగ్యాన్ని నిరంతరం మానిటర్ చేస్తుంది మరియు మీ అప్లికేషన్‌కు డౌన్‌టైమ్ లేకుండా అనారోగ్యకరమైన నోడ్‌లను స్వయంచాలకంగా భర్తీ చేస్తుంది. అదనంగా, Amazon MSK డేటాను విశ్రాంతి సమయంలో ఎన్‌క్రిప్ట్ చేయడం ద్వారా మీ Apache Kafka క్లస్టర్‌ను సురక్షితం చేస్తుంది.
