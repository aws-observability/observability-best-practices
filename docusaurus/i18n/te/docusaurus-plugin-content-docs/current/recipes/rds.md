# Amazon Relational Database Service

[Amazon Relational Database Service][rds-main] (RDS) క్లౌడ్‌లో రిలేషనల్ డేటాబేస్‌ను సెటప్ చేయడం, ఆపరేట్ చేయడం మరియు స్కేల్ చేయడం సులభం చేస్తుంది. ఇది హార్డ్‌వేర్ ప్రొవిజనింగ్, డేటాబేస్ సెటప్, ప్యాచింగ్ మరియు బ్యాకప్‌లు వంటి సమయం తీసుకునే నిర్వహణ పనులను ఆటోమేట్ చేస్తూ ఖర్చు-సమర్థమైన మరియు పునఃపరిమాణం చేయగల సామర్థ్యాన్ని అందిస్తుంది.

కింది రెసిపీలను చూడండి:

- [CloudWatch Logs, Lambda మరియు SNS తో RDS కోసం ప్రోయాక్టివ్ డేటాబేస్ మానిటరింగ్ నిర్మించడం][rds-cw-sns]
- [CloudWatch ఉపయోగించి PostgreSQL కోసం RDS మరియు PostgreSQL కోసం Aurora డేటాబేస్ లాగ్ ఎర్రర్‌లను మానిటర్ చేసి నోటిఫికేషన్‌లు సెటప్ చేయడం][rds-pg-au]
- [Amazon RDS లో లాగింగ్ మరియు మానిటరింగ్][rds-mon]
- [CloudWatch కు ప్రచురించబడిన Performance Insights మెట్రిక్స్][rds-pi-cw]
