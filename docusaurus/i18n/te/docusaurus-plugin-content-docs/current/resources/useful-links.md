# ఉపయోగకరమైన లింక్‌లు

AWS observability ఉత్తమ పద్ధతులను నేర్చుకోవడంలో మరియు అమలు చేయడంలో మీకు సహాయపడే క్యూరేటెడ్ రిసోర్స్‌ల సేకరణ.

### రాబోయే ఈవెంట్‌ల కోసం నమోదు చేసుకోండి (AWS Connected Community లో)
డెవలపర్లు, DevOps ఇంజనీర్లు, సొల్యూషన్స్ ఆర్కిటెక్ట్‌లు, సిస్టమ్ అడ్మినిస్ట్రేటర్లు, మరియు క్లౌడ్ ఆపరేషన్‌లలో ఆసక్తి ఉన్న ఎవరికైనా లైవ్ వెబినార్‌లు మరియు వర్క్‌షాప్‌ల కోసం [ఇక్కడ](https://aws-experience.com/amer/smb/events/series/Cloud-Operations-Enablement) నమోదు చేసుకోండి

### AWS Cloud Operations Show

ఆసక్తికరమైన AWS Cloud Operations ఫీచర్లు మరియు ఉపయోగ సందర్భాలలోకి లోతుగా అన్వేషించే లైవ్ స్ట్రీమ్ రికార్డింగ్‌ల సేకరణ. ఈ వీడియో సిరీస్ cloud operations లోకి ప్రాక్టికల్ డెమోన్‌స్ట్రేషన్‌లు మరియు నిపుణుల అంతర్దృష్టులను అందిస్తుంది.

[YouTube లో చూడండి](https://www.youtube.com/playlist?list=PLehXSATXjcQHj8bPSf0uZuQBoxJ7a7ag7)

### One Observability Workshop

ఈ hands-on వర్క్‌షాప్ AWS మానిటరింగ్ మరియు observability కోసం అందించే విస్తృత వైవిధ్యమైన టూల్ సెట్‌లతో ప్రాక్టికల్ అనుభవాన్ని అందిస్తుంది. వర్క్‌షాప్ అధిక అనుకూలీకరణ సాధ్యం - మీ ఆసక్తులు మరియు అందుబాటు ఆధారంగా మీరు ఏదైనా మాడ్యూల్‌ను ఎంచుకోవచ్చు.

[వర్క్‌షాప్ ప్రారంభించండి](https://catalog.workshops.aws/observability/)


### AWS Observability Best Practices గైడ్

ఈ Observability Best Practices గైడ్ యొక్క హోమ్ పేజీ. ఈ సమగ్ర రిసోర్స్ AWS పై observability ను అమలు చేయడానికి ఉత్తమ పద్ధతులు, ప్యాటర్న్‌లు, మరియు రెసిపీలను కవర్ చేస్తుంది.

[గైడ్ సందర్శించండి](https://aws-observability.github.io/observability-best-practices/)

### AWS Cloud Operations Best Practices గైడ్

Cloud Operations Best Practices గైడ్ యొక్క హోమ్ పేజీ. ఈ సమగ్ర రిసోర్స్ AWS పై మీ Cloud Operations వర్క్‌లోడ్‌లను కాన్ఫిగర్ చేయడం మరియు ఆపరేట్ చేయడం కోసం సిఫార్సు చేయబడిన విధానాలను కవర్ చేస్తుంది.

[గైడ్ సందర్శించండి](https://aws-samples.github.io/cloud-operations-best-practices/)

### AWS Distro for OpenTelemetry (ADOT)

ADOT తో మీ కోడ్‌ను ఎలా ఇన్‌స్ట్రుమెంట్ చేయాలనే దానిపై సమగ్ర గైడ్‌లు. AWS-మద్దతు ఉన్న OpenTelemetry డిస్ట్రిబ్యూషన్ ఉపయోగించి మీ అప్లికేషన్‌ల నుండి డిస్ట్రిబ్యూటెడ్ ట్రేసెస్ మరియు మెట్రిక్స్ ఎలా సేకరించాలో నేర్చుకోండి.

[ADOT డాక్యుమెంటేషన్ అన్వేషించండి](https://aws-otel.github.io/docs/introduction)

### AWS CloudTrail రిసోర్స్‌లు

మీ **AWS CloudTrail** భద్రతా పద్ధతులను బలపరచడానికి, incident response ను మెరుగుపరచడానికి, మరియు మానిటరింగ్ సామర్థ్యాలను పెంచడానికి క్యూరేటెడ్ రిసోర్స్‌లను అన్వేషించండి.

[గైడ్ సందర్శించండి](https://aws-samples.github.io/resources/cloudtrail-resources)

### Application Signals డెమో

AWS Application Signals సామర్థ్యాలను ప్రదర్శించే పూర్తి డెమోన్‌స్ట్రేషన్ అప్లికేషన్. ఈ డెమో ఒక పని చేసే ఉదాహరణ (Pet Clinic) ను అందిస్తుంది, దీన్ని మీరు Application Signals ఆచరణలో ఎలా పనిచేస్తుందో అర్థం చేసుకోవడానికి డిప్లాయ్ చేసి అన్వేషించవచ్చు.

[GitHub లో చూడండి](https://github.com/aws-observability/application-signals-demo)
