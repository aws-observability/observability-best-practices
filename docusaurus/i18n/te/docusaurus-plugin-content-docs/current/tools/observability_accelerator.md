# AWS Observability Accelerator

AWS Observability Accelerator అనేది Amazon Managed Service for Prometheus, Amazon Managed Grafana, AWS Distro for OpenTelemetry (ADOT) మరియు Amazon CloudWatch వంటి AWS నేటివ్ సేవలు మరియు AWS-నిర్వహిత observability సేవలతో మీ AWS పరిసరాల కోసం observability ను సెటప్ చేయడానికి సహాయపడే అభిప్రాయబద్ధమైన మాడ్యూల్‌ల సమూహం.

మీ EKS ఇన్‌ఫ్రాస్ట్రక్చర్, Java/JMX, NGINX ఆధారిత వర్క్‌లోడ్‌లు మరియు మీ కస్టమ్ అప్లికేషన్‌ల కోసం మేము క్యూరేటెడ్ మెట్రిక్స్, లాగ్‌లు, ట్రేసెస్ సేకరణ, CloudWatch డాష్‌బోర్డ్, అలర్టింగ్ రూల్స్ మరియు Grafana డాష్‌బోర్డ్‌లను అందిస్తాము.

AWS Observability Accelerator [Terraform](https://github.com/aws-observability/terraform-aws-observability-accelerator) మరియు [CDK](https://github.com/aws-observability/cdk-aws-observability-accelerator) ప్రాజెక్ట్‌ల కోసం షేర్డ్ ఆర్టిఫాక్ట్‌లను (డాక్యుమెంటేషన్, డాష్‌బోర్డ్‌లు, అలర్టింగ్ రూల్స్) అందిస్తుంది.

మరింత సమాచారం కోసం [Terraform](https://aws-observability.github.io/terraform-aws-observability-accelerator/) మరియు [CDK](https://aws-observability.github.io/cdk-aws-observability-accelerator/) ప్రాజెక్ట్‌ల డాక్యుమెంటేషన్‌ను చూడండి.
