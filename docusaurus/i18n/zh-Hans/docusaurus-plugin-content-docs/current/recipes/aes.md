# Amazon OpenSearch Service

[Amazon OpenSearch Service][aes-main] (AOS) 是 Amazon Elasticsearch Service 的后续产品，可以轻松执行交互式日志分析、实时应用程序监控、网站搜索等操作。OpenSearch 是一个源自 Elasticsearch 的开源分布式搜索和分析套件。它提供最新版本的 OpenSearch、支持 19 个版本的 Elasticsearch（1.5 到 7.10 版本），以及由 OpenSearch Dashboards 和 Kibana（1.5 到 7.10 版本）提供支持的可视化功能。
