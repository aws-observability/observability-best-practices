# EventBridge 与 JITNA 的集成

使用即时节点访问 (JITNA) 时，当创建需要手动批准的会话访问请求时，会向 [EventBridge](https://aws.amazon.com/eventbridge/) 发出三种类型的事件：

1. 请求者访问请求状态更新
1. 审批者访问请求状态更新
1. JITNA 访问请求失败

:::info
基于自动审批策略自动批准的访问请求不会发出事件。
:::
