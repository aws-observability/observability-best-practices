# Amazon DynamoDB

[Amazon DynamoDB][ddb-main] 是一种键值和文档数据库，能够在任何规模下提供个位数毫秒级的性能。它是一个完全托管的多区域、多活动、持久性数据库，内置安全性、备份和恢复以及内存缓存功能，适用于互联网规模的应用程序。
