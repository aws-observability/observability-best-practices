# Amazon Relational Database Service

[Amazon Relational Database Service][rds-main]（RDS）让您能够轻松地在云中设置、运行和扩展关系数据库。它提供经济高效且可调整大小的容量，同时自动化硬件配置、数据库设置、补丁更新和备份等耗时的管理任务。

请查看以下方案：

- [使用 CloudWatch Logs、Lambda 和 SNS 为 RDS 构建主动数据库监控][rds-cw-sns]
- [监控 RDS for PostgreSQL 和 Aurora for PostgreSQL 数据库日志错误，并使用 CloudWatch 设置通知][rds-pg-au]
- [Amazon RDS 中的日志记录与监控][rds-mon]
- [发布到 CloudWatch 的 Performance Insights metrics][rds-pi-cw]
