# AWS Observability Accelerator

AWS Observability Accelerator 是一组经过精心设计的模块，帮助您使用 AWS 原生服务和 AWS 托管可观测性服务（如 Amazon Managed Service for Prometheus、Amazon Managed Grafana、AWS Distro for OpenTelemetry (ADOT) 和 Amazon CloudWatch）为您的 AWS 环境设置 observability。

我们为您的 EKS 基础设施、Java/JMX、基于 NGINX 的工作负载以及自定义应用程序提供精选的 metrics、logs、traces 采集、CloudWatch dashboard、告警规则和 Grafana dashboards。

AWS Observability Accelerator 为 [Terraform](https://github.com/aws-observability/terraform-aws-observability-accelerator) 和 [CDK](https://github.com/aws-observability/cdk-aws-observability-accelerator) 项目提供共享资源（文档、dashboards、告警规则）。

请查阅 [Terraform](https://aws-observability.github.io/terraform-aws-observability-accelerator/) 和 [CDK](https://aws-observability.github.io/cdk-aws-observability-accelerator/) 项目的文档以获取更多信息。
