Surveillance des utilisateurs reels (RUM)

Comment RUM se connecte a Application Signals

Les clients web CloudWatch RUM peuvent apparaitre dans Application Signals lorsque vous activez le tracage X-Ray pour le client RUM. Les requetes du client vers les services backend sont alors associees et affichees dans les details du service et sur la Carte d'application.

Voir : Comment Application Signals permet la surveillance de la sante operationnelle.

A quelles questions RUM aide a repondre

  • Ce probleme impacte-t-il les utilisateurs ? Correlez la sante du backend aux signaux de chargement de page et d'erreurs JS
  • Ou les utilisateurs sont-ils impactes ? Comparez les performances par geographie, appareil et navigateur
  • Qu'est-ce qui a change ? Correlez avec les deploiements/evenements de changement lors de l'investigation des regressions
