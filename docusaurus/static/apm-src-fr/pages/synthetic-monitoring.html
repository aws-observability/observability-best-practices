Surveillance synthétique (Canaries)

Comment Synthetics se connecte à Application Signals

Les canaries CloudWatch Synthetics peuvent apparaître dans Application Signals lorsque vous activez le traçage X-Ray pour le canary. Cela permet aux appels des canaries d'être associés aux services backend et affichés dans les détails du service et sur la carte des applications.

Voir : Comment Application Signals permet la surveillance de la santé opérationnelle.

Cas d'utilisation courants des canaries

  • Vérifications de santé des API : valider les points de terminaison clés selon un calendrier
  • Flux de navigation : valider les parcours de connexion/paiement
  • Multi-région : détecter les problèmes régionaux avant les clients
