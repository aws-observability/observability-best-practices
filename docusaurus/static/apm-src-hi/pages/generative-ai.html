CloudWatch APM दस्तावेज़ीकरण

जनरेटिव AI observability

Application Signals + Amazon Bedrock

CloudWatch Application Signals Amazon Bedrock मॉडल के साथ इंटरैक्ट करने वाले जनरेटिव AI एप्लिकेशन की समस्या निवारण का समर्थन करता है। यह Bedrock API कॉल के लिए आउट-ऑफ़-द-बॉक्स टेलीमेट्री प्रदान करता है ताकि आप इनकी जांच कर सकें:

  • मॉडल कॉन्फ़िगरेशन समस्याएं
  • उपयोग और लागत कारक (टोकन उपयोग)
  • मॉडल लेटेंसी
  • रुकने/समाप्त होने के कारण प्रतिक्रियाओं के लिए

विस्तृत मेट्रिक्स और ट्रेसेज़

Bedrock कॉल के लिए, Application Signals रिसोर्स-लेवल विवरण (उदाहरण के लिए: मॉडल ID, गार्डरेल ID, नॉलेज बेस ID, और Bedrock एजेंट ID) के साथ प्रदर्शन मेट्रिक्स और ट्रेस स्पैन जनरेट और सहसंबंधित कर सकता है।

देखें: उदाहरण: Amazon Bedrock के साथ इंटरैक्ट करने वाले जनरेटिव AI ऐप्स की समस्या निवारण।

GenAI सिमैंटिक एट्रिब्यूट + Transaction Search

Application Signals OpenTelemetry GenAI सिमैंटिक कन्वेंशन एट्रिब्यूट (उदाहरण के लिए, gen_ai.request.model और टोकन उपयोग फ़ील्ड) उत्सर्जित कर सकता है। आप मॉडल में प्रॉम्प्ट, टोकन उपयोग और लेटेंसी की तुलना करने के लिए CloudWatch Transaction Search के साथ इनका विश्लेषण कर सकते हैं।

  • Transaction Search
  • स्पैन खोजना और विश्लेषण करना
