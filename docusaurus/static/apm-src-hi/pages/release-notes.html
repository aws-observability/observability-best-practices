रिलीज़ नोट्स: कहां देखें

CloudWatch Application Signals एक प्रबंधित AWS क्षमता है। सार्वजनिक रिलीज़ नोट्स और फ़ीचर घोषणाएं AWS दस्तावेज़ीकरण और AWS "What's New" के माध्यम से प्रकाशित की जाती हैं।

  • AWS "What's New": aws.amazon.com/about-aws/whats-new
  • Application Signals डॉक्स: CloudWatch Application Signals

इंस्ट्रूमेंटेशन कंपोनेंट

आपके "वर्ज़न्ड" अपग्रेड आमतौर पर इंस्ट्रूमेंटेशन डिस्ट्रीब्यूशन और उसकी डिपेंडेंसीज़ (उदाहरण के लिए, AWS Distro for OpenTelemetry और OpenTelemetry इंस्ट्रूमेंटेशन लाइब्रेरी), साथ ही CloudWatch Agent / प्लेटफ़ॉर्म ऐड-ऑन से आते हैं।

  • ADOT: अपनी भाषा/रनटाइम के लिए उपयुक्त AWS Distro for OpenTelemetry रिलीज़ नोट्स का पालन करें।
  • CloudWatch Agent: नई Application Signals क्षमताओं (उदाहरण के लिए, रिसोर्स एट्रिब्यूट के माध्यम से ग्रुपिंग) के लिए आवश्यक होने पर एजेंट को अपग्रेड करें।

ज्ञात समस्याएं और संगतता

वर्तमान संगतता मैट्रिक्स और ज्ञात समस्याओं के लिए, आधिकारिक "समर्थित सिस्टम" पेज देखें।
