जून 2024: Application Signals सामान्य उपलब्धता

CloudWatch Application Signals सामान्य रूप से उपलब्ध (GA) हो गया। यह कॉल वॉल्यूम, उपलब्धता, लेटेंसी, फॉल्ट और एरर जैसे मानक एप्लिकेशन मेट्रिक्स को स्वचालित रूप से एकत्र करके त्वरित सर्विस-लेवल ट्राइएज और समस्या निवारण पर केंद्रित है।

AWS घोषणा देखें: GA घोषणा।

सितंबर 2024: रिक्वेस्ट-बेस्ड SLO

AWS ने CloudWatch Application Signals के लिए रिक्वेस्ट-बेस्ड सर्विस लेवल ऑब्जेक्टिव (SLO) सपोर्ट की घोषणा की।

AWS घोषणा देखें: रिक्वेस्ट-बेस्ड SLO।

जारी: Application Map और ट्राइएज वर्कफ़्लो

CloudWatch Application Map Application Signals के लिए "Service Map" शब्दावली को प्रतिस्थापित करता है। यह टोपोलॉजी (क्लाइंट, कैनरी, सर्विसेज़, डिपेंडेंसीज़) को विज़ुअलाइज़ करता है और टीमों को इंसिडेंट ट्राइएज में मदद करने के लिए SLO/SLI स्वास्थ्य प्रदर्शित कर सकता है।

डॉक्स देखें: Application Map।
