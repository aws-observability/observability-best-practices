CloudWatch APM Documentation

ようこそ

このドキュメントサイトでは、本番環境におけるアプリケーションの動作を把握する必要がある開発者、オペレーター、およびアーキテクト向けに、CloudWatch Application Performance Monitoring (APM) について説明します。

APM は、サービスの健全性（レイテンシー、エラー、可用性、スループット）をトレースおよびアーキテクチャ全体の関連シグナルと関連付けることで、検出までの時間と解決までの時間を短縮するのに役立ちます。

左側のナビゲーションを使用して、実装ガイダンス、主要機能、およびトラブルシューティングのワークフローを確認してください。

使い始めるには、 Getting Started ガイドをご覧ください。
