Application Signals + Amazon Bedrock

CloudWatch Application Signals は、Amazon Bedrock モデルと連携する生成 AI アプリケーションのトラブルシューティングをサポートします。Bedrock API コールに対してすぐに使えるテレメトリを提供し、以下を調査できます：

  • モデル設定の問題
  • 使用状況とコストの要因 （トークン使用量）
  • モデルのレイテンシー
  • 停止/終了理由 レスポンスに対して

きめ細かいメトリクスとトレース

Bedrock コールに対して、Application Signals はパフォーマンスメトリクスとトレーススパンをリソースレベルの詳細（例：モデル ID、ガードレール ID、ナレッジベース ID、Bedrock エージェント ID）と関連付けて生成できます。

参照： 例：Amazon Bedrock と連携する生成 AI アプリのトラブルシューティング.

GenAI セマンティック属性 + Transaction Search

Application Signals は OpenTelemetry GenAI セマンティック規約の属性（例： gen_ai.request.model およびトークン使用量フィールド）を出力できます。これらを CloudWatch Transaction Search と組み合わせて、モデル間のプロンプト、トークン使用量、レイテンシーを比較分析できます。
