カスタムメトリクス（apm-center サンプルより）

以下のサンプルサービスの一部には、 apm-center テスト中に「カスタムメトリクス」の動作を検証するために使用するエンドポイントが含まれています。たとえば、EKS Python サンプルには次のものが含まれています GET /api/metrics:

EKS Python 有効化ガイドを開く

@app.get("/api/metrics")
async def metrics():
    """Custom metrics endpoint"""
    # Simulate some business logic
    processing_time = random.uniform(0.1, 0.5)
    time.sleep(processing_time)

    # Simulate different response times based on some logic
    if processing_time > 0.3:
        status = "slow"
    else:
        status = "fast"

    return {
        "message": "Metrics endpoint",
        "processing_time": processing_time,
        "status": status,
        "timestamp": time.time(),
        "service": "python-eks-service"
    }
