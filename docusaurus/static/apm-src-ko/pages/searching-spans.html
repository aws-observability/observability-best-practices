스팬 검색 및 분석

Transaction Search

CloudWatch Transaction Search는 스팬 속성을 사용하여 수집된 스팬을 검색하고 분석하는 대화형 분석 경험입니다. 지원 기능:

  • 목록: 개별 트랜잭션 문제 해결, 느린 스팬 찾기, 오류 찾기
  • 시계열: 지연 시간 및 빈도 추세 시각화 (P50/P90/P99)
  • 그룹 분석: 속성별 집계 (예: 영향을 받는 상위 고객)

참조: 스팬 검색 및 분석.

스팬에 대한 Logs Insights

CloudWatch Logs Insights를 사용하여 스팬을 쿼리할 수도 있습니다(예: DB 문별 p99 지속 시간, 또는 HTTP 5xx가 있는 스팬 수).

참조: Transaction Search.
