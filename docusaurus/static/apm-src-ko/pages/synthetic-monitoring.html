합성 모니터링 (카나리)

Synthetics가 Application Signals에 연결되는 방식

CloudWatch Synthetics 카나리는 카나리에 대해 X-Ray 트레이싱을 활성화하면 Application Signals에 표시될 수 있습니다. 이를 통해 카나리 호출이 백엔드 서비스와 연결되어 서비스 세부 정보와 Application Map에 표시됩니다.

참조: Application Signals가 운영 상태 모니터링을 가능하게 하는 방법.

일반적인 카나리 사용 사례

  • API 상태 확인: 일정에 따라 주요 엔드포인트를 검증합니다
  • 브라우저 흐름: 로그인/결제 여정을 검증합니다
  • 다중 리전: 고객보다 먼저 리전별 문제를 감지합니다
