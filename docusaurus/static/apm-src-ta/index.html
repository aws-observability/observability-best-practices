CloudWatch APM ஆவணங்கள்

வரவேற்கிறோம்

இந்த ஆவண தளம் CloudWatch Application Performance Monitoring (APM) பற்றிய தகவல்களை உற்பத்தியில் பயன்பாட்டு நடத்தையைப் புரிந்துகொள்ள வேண்டிய டெவலப்பர்கள், ஆபரேட்டர்கள் மற்றும் ஆர்கிடெக்ட்களுக்காக வழங்குகிறது.

APM உங்கள் கட்டமைப்பு முழுவதும் சேவை ஆரோக்கியத்தை (தாமதம், பிழைகள், கிடைக்கும் தன்மை மற்றும் செயல்திறன்) traces மற்றும் தொடர்புடைய சிக்னல்களுடன் தொடர்புபடுத்துவதன் மூலம் கண்டறிய-ஆகும்-நேரம் மற்றும் தீர்க்க-ஆகும்-நேரத்தை குறைக்க உதவுகிறது.

செயல்படுத்தல் வழிகாட்டுதல்கள், முக்கிய அம்சங்கள் மற்றும் சிக்கல் தீர்வு பணிப்பாய்வுகளை ஆராய இடது வழிசெலுத்தலைப் பயன்படுத்தவும்.

தொடங்குதல் வழிகாட்டியுடன் ஆரம்பிக்கவும்.
