CloudWatch Application Signals ஒரு managed AWS திறன் ஆகும். பொது வெளியீட்டு குறிப்புகள் மற்றும் அம்ச அறிவிப்புகள் AWS documentation மற்றும் AWS "What's New" வழியாக publish செய்யப்படுகின்றன.

  • AWS "What's New": aws.amazon.com/about-aws/whats-new
  • Application Signals docs: CloudWatch Application Signals

Instrumentation components

உங்கள் "versioned" upgrades பொதுவாக instrumentation distribution மற்றும் அதன் dependencies-லிருந்து (எடுத்துக்காட்டாக, AWS Distro for OpenTelemetry மற்றும் OpenTelemetry instrumentation libraries), மற்றும் CloudWatch Agent / platform add-ons-லிருந்து வருகின்றன.

  • ADOT: உங்கள் language/runtime-க்கு பொருத்தமான AWS Distro for OpenTelemetry release notes-ஐ பின்தொடரவும்.
  • CloudWatch Agent: புதிய Application Signals திறன்களுக்கு (எடுத்துக்காட்டாக, resource attributes வழியாக grouping) தேவைப்படும்போது agent-ஐ upgrade செய்யுங்கள்.
