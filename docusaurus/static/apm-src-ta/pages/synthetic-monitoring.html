Synthetic Monitoring (Canaries)

Synthetics Application Signals-உடன் எவ்வாறு இணைகிறது

Canary-க்கு X-Ray tracing-ஐ இயக்கும்போது CloudWatch Synthetics canaries Application Signals-ல் தோன்றலாம். இது canary அழைப்புகளை backend சேவைகளுடன் தொடர்புபடுத்தி சேவை விவரங்கள் மற்றும் Application Map-ல் காட்ட அனுமதிக்கிறது.

பார்க்கவும்: Application Signals செயல்பாட்டு நிலை கண்காணிப்பை எவ்வாறு இயக்குகிறது.

பொதுவான canary பயன்பாட்டு நிலைகள்

  • API health checks: ஒரு அட்டவணையில் முக்கிய endpoints-ஐ validate செய்தல்
  • Browser flows: login/checkout பயணங்களை validate செய்தல்
  • Multi-region: வாடிக்கையாளர்களுக்கு முன் regional சிக்கல்களை கண்டறிதல்
