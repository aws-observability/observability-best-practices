CloudWatch APM డాక్యుమెంటేషన్

Span లను శోధించడం మరియు విశ్లేషించడం

Transaction Search

CloudWatch Transaction Search అనేది span అట్రిబ్యూట్‌లను ఉపయోగించి ఇన్‌జెస్ట్ చేయబడిన span లను శోధించడం మరియు విశ్లేషించడం కోసం ఇంటరాక్టివ్ అనలిటిక్స్ అనుభవం. ఇది మద్దతు ఇస్తుంది:

  • జాబితా: వ్యక్తిగత లావాదేవీలను ట్రబుల్‌షూట్ చేయండి, నెమ్మదిగా ఉన్న span లను కనుగొనండి, ఎర్రర్‌లను కనుగొనండి
  • టైమ్ సీరీస్: లేటెన్సీ మరియు ఫ్రీక్వెన్సీ ట్రెండ్‌లను దృశ్యీకరించండి (P50/P90/P99)
  • గ్రూప్ విశ్లేషణ: అట్రిబ్యూట్‌ల ద్వారా సమగ్రపరచండి (ఉదాహరణకు, ప్రభావిత అగ్ర కస్టమర్‌లు)

చూడండి: Span లను శోధించడం మరియు విశ్లేషించడం.

Span లపై Logs Insights

మీరు CloudWatch Logs Insights ఉపయోగించి span లను కూడా క్వెరీ చేయవచ్చు (ఉదాహరణకు, DB స్టేట్‌మెంట్ ద్వారా p99 వ్యవధి, లేదా HTTP 5xx ఉన్న span ల సంఖ్యలు).

చూడండి: Transaction Search.
