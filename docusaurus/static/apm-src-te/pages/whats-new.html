కొత్తగా ఏముంది

ఈ పేజీ ఏమి కవర్ చేస్తుంది

ఈ సైట్ స్టాటిక్ డాక్స్ స్కాఫోల్డ్. కంటెంట్‌ను ఖచ్చితంగా ఉంచడానికి, ఇక్కడ "కొత్తగా ఏముంది" గమనార్హమైన పబ్లిక్ మైలురాళ్లను సంగ్రహిస్తుంది మరియు వివరాల కోసం అధికారిక AWS మూలాలను సూచిస్తుంది.

  • AWS ప్రకటనలు: AWS "What's New"
  • CloudWatch డాక్స్: Application Signals

జూన్ 2024: Application Signals సాధారణ అందుబాటు

CloudWatch Application Signals సాధారణంగా అందుబాటులోకి (GA) వచ్చింది. ఇది కాల్ వాల్యూమ్, అందుబాటు, లేటెన్సీ, ఫాల్ట్‌లు మరియు ఎర్రర్‌లు వంటి ప్రామాణిక అప్లికేషన్ మెట్రిక్స్‌ను స్వయంచాలకంగా సేకరించడం ద్వారా వేగవంతమైన సేవ-స్థాయి ట్రయేజ్ మరియు ట్రబుల్‌షూటింగ్‌పై దృష్టి పెడుతుంది.

AWS ప్రకటన చూడండి: GA ప్రకటన.

సెప్టెంబర్ 2024: అభ్యర్థన-ఆధారిత SLOలు

AWS CloudWatch Application Signals కోసం అభ్యర్థన-ఆధారిత సేవా స్థాయి లక్ష్యాలు (SLOs) మద్దతును ప్రకటించింది.

AWS ప్రకటన చూడండి: అభ్యర్థన-ఆధారిత SLOలు.

కొనసాగుతోంది: అప్లికేషన్ మ్యాప్ మరియు ట్రయేజ్ వర్క్‌ఫ్లో

CloudWatch అప్లికేషన్ మ్యాప్ Application Signals కోసం "సర్వీస్ మ్యాప్" పదజాలాన్ని భర్తీ చేస్తుంది. ఇది టోపాలజీ (క్లయింట్‌లు, కానరీలు, సేవలు, డిపెండెన్సీలు) ను దృశ్యీకరిస్తుంది మరియు సంఘటనలను ట్రయేజ్ చేయడంలో బృందాలకు సహాయపడటానికి SLO/SLI ఆరోగ్యాన్ని ప్రదర్శించగలదు.

డాక్స్ చూడండి: అప్లికేషన్ మ్యాప్.
