概述

本指南提供分步说明，指导您部署启用 AWS Application Signals 的 .NET Lambda 函数。这是使用 AWS Lambda 控制台内置层支持的最简单方法。

完成目标：
  • 部署启用 Application Signals 的 .NET Lambda 函数
  • 通过 Lambda Layer 启用 AWS Application Signals 自动插桩
  • 在 CloudWatch Application Signals 控制台中查看您的服务
  • 监控服务 metrics、traces 和操作
完成时间： 15-20 分钟

前置条件

必备工具

  • AWS CLI v2（最新版本）
  • .NET 8 SDK
  • 具有适当权限的 AWS 账户
  • 文本编辑器

所需 AWS 权限

  • Lambda：创建和管理函数
  • IAM：创建角色和策略
  • CloudWatch：Logs 和 Application Signals 访问
  • S3：基本读取权限（用于测试）

架构

您的 Lambda 函数将通过以下方式自动插桩：

  1. AWS Lambda Layer for OpenTelemetry - 提供自动插桩
  2. Application Signals 集成 - 内置于层中
  3. 自动追踪 - HTTP 请求、AWS SDK 调用、函数执行

步骤 1： 启用 Application Signals in Your Account

重要： 每个 AWS 账户每个区域只需启用一次 Application Signals。

方案 1：通过 AWS 控制台启用（推荐）

  1. 前往 AWS 控制台 → CloudWatch → Application Signals
  2. 选择您的区域（例如 us-east-1)
  3. Click "Get Started" or "启用 Application Signals"
  4. 等待 2-3 分钟激活

2.3 Create Program.cs

Create the main Lambda function code:

using Amazon.Lambda.APIGatewayEvents;
using Amazon.Lambda.Core;
using Amazon.Lambda.RuntimeSupport;
using Amazon.Lambda.Serialization.SystemTextJson;

namespace lambda_dotnet;

/// <summary>
/// Simple Hello World Lambda function for Application Signals testing
/// </summary>
public class Program
{
    private static async Task Main()
    {
        using var handlerWrapper = HandlerWrapper.GetHandlerWrapper<APIGatewayProxyRequest, APIGatewayProxyResponse>(FunctionHandlerAsync, new DefaultLambdaJsonSerializer());
        using var bootstrap = new LambdaBootstrap(handlerWrapper);
        await bootstrap.RunAsync();
    }

    /// <summary>
    /// Simple Lambda function handler
    /// </summary>
    public static async Task<APIGatewayProxyResponse> FunctionHandlerAsync(APIGatewayProxyRequest request, ILambdaContext context)
    {
        context.Logger.LogInformation($"Received event: {System.Text.Json.JsonSerializer.Serialize(request)}");

        try
        {
            var path = request.Path ?? "/";
            var httpMethod = request.HttpMethod ?? "GET";

            object response;

            if (path == "/" || path == "")
            {
                response = new
                {
                    message = "Hello World from .NET Lambda!",
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    function_name = context.FunctionName,
                    request_id = context.AwsRequestId,
                    service = "lambda-dotnet",
                    runtime = "dotnet8"
                };
            }
            else if (path == "/health")
            {
                response = new
                {
                    status = "healthy",
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    function_name = context.FunctionName,
                    runtime = "dotnet8"
                };
            }
            else if (path.StartsWith("/api/hello/"))
            {
                var name = path.Substring("/api/hello/".Length);
                response = new
                {
                    message = $"Hello, {name}!",
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    function_name = context.FunctionName,
                    runtime = "dotnet8"
                };
            }
            else
            {
                response = new
                {
                    error = "Not Found",
                    path = path,
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds()
                };
            }

            return new APIGatewayProxyResponse
            {
                StatusCode = 200,
                Headers = new Dictionary<string, string>
                {
                    { "Content-Type", "application/json" },
                    { "Access-Control-Allow-Origin", "*" }
                },
                Body = System.Text.Json.JsonSerializer.Serialize(response)
            };
        }
        catch (Exception ex)
        {
            context.Logger.LogError($"Error processing request: {ex}");

            return new APIGatewayProxyResponse
            {
                StatusCode = 500,
                Headers = new Dictionary<string, string>
                {
                    { "Content-Type", "application/json" }
                },
                Body = System.Text.Json.JsonSerializer.Serialize(new
                {
                    error = "Internal Server Error",
                    message = ex.Message,
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    function_name = context.FunctionName
                })
            };
        }
    }
}

2.4 Create Controllers/DemoController.cs

使用各种端点创建控制器：

using Amazon.Lambda.Core;
using Amazon.S3;
using Amazon.S3.Model;

namespace lambda_dotnet.Controllers;

/// <summary>
/// Demo controller with various endpoints for testing Application Signals
/// 
/// All endpoints are automatically instrumented by the AWS Lambda Layer for OpenTelemetry.
/// No manual tracing code is required.
/// </summary>
public class DemoController
{
    private readonly Random _random = new();
    private readonly HttpClient _httpClient;
    private readonly IAmazonS3 _s3Client;

    public DemoController()
    {
        _httpClient = new HttpClient
        {
            Timeout = TimeSpan.FromSeconds(10)
        };
        _s3Client = new AmazonS3Client();
    }

    /// <summary>
    /// Root endpoint
    /// </summary>
    public async Task<object> RootAsync(ILambdaContext context)
    {
        return new
        {
            message = "AWS Application Signals Lambda .NET Demo",
            timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
            function_name = context.FunctionName,
            request_id = context.AwsRequestId,
            service = "lambda-dotnet",
            runtime = "dotnet8"
        };
    }

    /// <summary>
    /// Health check endpoint
    /// </summary>
    public async Task<object> HealthAsync(ILambdaContext context)
    {
        return new
        {
            status = "healthy",
            timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
            function_name = context.FunctionName,
            memory_limit = context.MemoryLimitInMB,
            remaining_time = context.RemainingTime,
            runtime = "dotnet8"
        };
    }

    /// <summary>
    /// Personalized greeting with simulated processing time
    /// </summary>
    public async Task<object> HelloAsync(string name, ILambdaContext context)
    {
        // Simulate some processing time
        await Task.Delay(_random.Next(10, 110)); // 10-110ms

        return new
        {
            message = $"Hello, {name}!",
            timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
            function_name = context.FunctionName,
            greeting_id = _random.Next(10000),
            runtime = "dotnet8"
        };
    }

    /// <summary>
    /// Intentionally slow endpoint for latency testing
    /// </summary>
    public async Task<object> SlowAsync(ILambdaContext context)
    {
        // Simulate slow operation (500-1500ms)
        var delay = _random.Next(500, 1500);
        await Task.Delay(delay);

        return new
        {
            message = "Slow operation completed",
            timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
            function_name = context.FunctionName,
            duration_ms = delay,
            runtime = "dotnet8"
        };
    }

    /// <summary>
    /// Endpoint that randomly raises errors for fault testing
    /// </summary>
    public async Task<object> ErrorAsync(ILambdaContext context)
    {
        if (_random.NextDouble() > 0.5)
        {
            throw new Exception("Simulated error for Application Signals testing");
        }

        return new
        {
            message = "No error occurred",
            timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
            function_name = context.FunctionName,
            runtime = "dotnet8"
        };
    }

    /// <summary>
    /// Makes external HTTP call for distributed tracing
    /// </summary>
    public async Task<object> ExternalAsync(ILambdaContext context)
    {
        try
        {
            var response = await _httpClient.GetAsync("https://httpbin.org/delay/1");

            return new
            {
                message = "External call completed",
                external_status_code = (int)response.StatusCode,
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                function_name = context.FunctionName,
                runtime = "dotnet8"
            };
        }
        catch (Exception ex)
        {
            throw new Exception($"External call failed: {ex.Message}");
        }
    }

    /// <summary>
    /// AWS S3 call for testing AWS SDK instrumentation
    /// </summary>
    public async Task<object> S3Async(ILambdaContext context)
    {
        try
        {
            // List buckets (this is a safe operation that doesn't require specific permissions)
            var response = await _s3Client.ListBucketsAsync();

            var result = new
            {
                message = "S3 call completed successfully",
                bucket_count = response.Buckets.Count,
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                function_name = context.FunctionName,
                runtime = "dotnet8"
            };

            // Add first bucket name if available
            if (response.Buckets.Count > 0)
            {
                result = new
                {
                    message = "S3 call completed successfully",
                    bucket_count = response.Buckets.Count,
                    first_bucket = response.Buckets[0].BucketName,
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    function_name = context.FunctionName,
                    runtime = "dotnet8"
                };
            }

            return result;
        }
        catch (Exception ex)
        {
            return new
            {
                error = "S3 call failed",
                message = ex.Message,
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                function_name = context.FunctionName,
                runtime = "dotnet8"
            };
        }
    }

    /// <summary>
    /// Database simulation endpoint
    /// </summary>
    public async Task<object> DatabaseAsync(ILambdaContext context)
    {
        // Simulate database query time
        await Task.Delay(_random.Next(50, 250)); // 50-250ms

    /// Complex operation with multiple steps
    /// </summary>
    public async Task<object> ComplexAsync(ILambdaContext context)

步骤 4： 为 Lambda 创建 IAM 角色

使用以下策略创建 IAM 角色：

  • AWSLambdaBasicExecutionRole
  • CloudWatchLambdaApplicationSignalsExecutionRolePolicy (CRITICAL!)

4.1 创建 IAM 角色

aws --profile eksworkstream --region us-east-1 iam create-role \
  --role-name lambda-dotnet-execution-role \
  --assume-role-policy-document '{
    "Version": "2012-10-17",
    "Statement": [{
      "Effect": "Allow",
      "Principal": {"Service": "lambda.amazonaws.com"},
      "Action": "sts:AssumeRole"
    }]
  }'

4.2 附加所需策略

aws --profile eksworkstream --region us-east-1 iam attach-role-policy \
  --role-name lambda-dotnet-execution-role \
  --policy-arn arn:aws:iam::aws:policy/service-role/AWSLambdaBasicExecutionRole

aws --profile eksworkstream --region us-east-1 iam attach-role-policy \
  --role-name lambda-dotnet-execution-role \
  --policy-arn arn:aws:iam::aws:policy/CloudWatchLambdaApplicationSignalsExecutionRolePolicy

4.3 获取角色 ARN

ROLE_ARN=$(aws --profile eksworkstream --region us-east-1 iam get-role \
  --role-name lambda-dotnet-execution-role \
  --query 'Role.Arn' \
  --output text)

echo "Role ARN: $ROLE_ARN"

Step 7: 生成流量 for Testing

To test Application Signals, you need to generate traffic to your Lambda function. We provide both Python and shell script traffic generators:

7.1 Python Traffic Generator

创建 traffic-generator.py file:

#!/usr/bin/env python3

"""
.NET Lambda Traffic Generator for Application Signals Testing
Generates various types of requests to test Application Signals metrics and traces
"""

import boto3
import json
import time
import random
import sys
from datetime import datetime

def generate_traffic(function_name, region='us-east-1', profile='eksworkstream', num_requests=50):
    """
    Generate traffic to the Lambda function for Application Signals testing
    """
    print(f"Starting traffic generation for {function_name}")
    print(f"   Region: {region}")
    print(f"   Profile: {profile}")
    print(f"   Requests: {num_requests}")
    print("-" * 50)

    # Initialize Lambda client
    try:
        session = boto3.Session(profile_name=profile)
        lambda_client = session.client('lambda', region_name=region)
        print("AWS Lambda client initialized")
    except Exception as e:
        print(f"Error initializing AWS client: {e}")
        return False

    # Test endpoints with different characteristics
    endpoints = [
        {"path": "/", "method": "GET", "name": "Root", "weight": 20},
        {"path": "/health", "method": "GET", "name": "Health", "weight": 15},
        {"path": "/api/hello/User", "method": "GET", "name": "Hello", "weight": 30},
        {"path": "/api/slow", "method": "GET", "name": "Slow", "weight": 10},
        {"path": "/api/error", "method": "GET", "name": "Error", "weight": 10},
        {"path": "/api/external", "method": "GET", "name": "External", "weight": 8},
        {"path": "/api/s3", "method": "GET", "name": "S3", "weight": 5},
        {"path": "/api/database", "method": "GET", "name": "Database", "weight": 2},
    ]

    # Create weighted list for random selection
    weighted_endpoints = []
    for endpoint in endpoints:
        weighted_endpoints.extend([endpoint] * endpoint["weight"])

    success_count = 0
    error_count = 0

    print(f"Generating {num_requests} requests...")
    print()

    for i in range(1, num_requests + 1):
        # Select random endpoint
        endpoint = random.choice(weighted_endpoints)

        if endpoint["name"] == "Hello":
            user_name = f"User{random.randint(1, 1000)}"
            path = f"/api/hello/{user_name}"
        else:
            path = endpoint["path"]

        payload = {"path": path, "httpMethod": endpoint["method"]}

        start_time = time.time()
        try:
            response = lambda_client.invoke(
                FunctionName=function_name,
                InvocationType='RequestResponse',
                Payload=json.dumps(payload)
            )
            end_time = time.time()

            # Parse response
            response_payload = json.loads(response['Payload'].read())
            status_code = response_payload.get('statusCode', 200)
            duration = round((end_time - start_time) * 1000, 2)

            if status_code == 200:
                success_count += 1
                status_icon = "SUCCESS"
            else:
                error_count += 1
                status_icon = "WARNING"

            # Print progress
            print(f"{status_icon} Request {i:2d}: {endpoint['name']:8s} | "
                  f"Status: {status_code} | Duration: {duration:6.2f}ms | "
                  f"Path: {path}")

            # Add small delay between requests
            time.sleep(random.uniform(0.1, 0.5))

        except Exception as e:
            error_count += 1
            print(f"ERROR Request {i:2d}: {endpoint['name']:8s} | Error: {str(e)[:50]}...")
            time.sleep(0.5)

    # Print summary
    print()
    print("=" * 50)
    print("TRAFFIC GENERATION SUMMARY")
    print("=" * 50)
    print(f"Total Requests: {num_requests}")
    print(f"Successful:     {success_count} ({success_count/num_requests*100:.1f}%)")
    print(f"Errors:         {error_count} ({error_count/num_requests*100:.1f}%)")
    print(f"Timestamp:      {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print()
    print("Next Steps:")
    print("1. Wait 5-10 minutes for Application Signals to process data")
    print("2. Check Application Signals console:")
    print("   https://us-east-1.console.aws.amazon.com/cloudwatch/home?region=us-east-1#application-signals:services")
    print("3. Look for 'lambda-dotnet' in the services list")
    print()

    return success_count > 0

def main():
    """Main function"""
    if len(sys.argv) < 2:
        print("Usage: python3 traffic-generator.py  [num_requests] [region] [profile]")
        sys.exit(1)

    function_name = sys.argv[1]
    num_requests = int(sys.argv[2]) if len(sys.argv) > 2 else 50
    region = sys.argv[3] if len(sys.argv) > 3 else 'us-east-1'
    profile = sys.argv[4] if len(sys.argv) > 4 else 'eksworkstream'

    success = generate_traffic(function_name, region, profile, num_requests)

    if success:
        print("Traffic generation completed successfully!")
        sys.exit(0)
    else:
        print("Traffic generation failed!")
        sys.exit(1)

if __name__ == "__main__":
    main()

7.2 Shell 脚本流量生成器

用于流量生成的替代 shell 脚本：

#!/bin/bash

# .NET Lambda Traffic Generator for Application Signals Testing
# Generates various types of requests to test Application Signals metrics and traces

FUNCTION_NAME="lambda-dotnet"
NUM_REQUESTS=50
REGION="us-east-1"
PROFILE="eksworkstream"

# Parse arguments
if [ $# -ge 1 ]; then
    FUNCTION_NAME="$1"
fi
if [ $# -ge 2 ]; then
    NUM_REQUESTS="$2"
fi
if [ $# -ge 3 ]; then
    REGION="$3"
fi
if [ $# -ge 4 ]; then
    PROFILE="$4"
fi

echo "Starting traffic generation for ${FUNCTION_NAME}"
echo "   Region: ${REGION}"
echo "   Profile: ${PROFILE}"
echo "   Requests: ${NUM_REQUESTS}"
echo "--------------------------------------------------"

# Test endpoints with different characteristics
declare -a ENDPOINTS=(
    "root:/:GET:20"
    "health:/health:GET:15"
    "hello:/api/hello:GET:30"
    "slow:/api/slow:GET:10"
    "error:/api/error:GET:10"
    "external:/api/external:GET:8"
    "s3:/api/s3:GET:5"
    "database:/api/database:GET:2"
)

success_count=0
error_count=0

echo "Generating ${NUM_REQUESTS} requests..."
echo

for i in $(seq 1 $NUM_REQUESTS); do
    # Select random endpoint
    endpoint_info=${ENDPOINTS[$((RANDOM % ${#ENDPOINTS[@]}))]}
    IFS=':' read -r name path method weight <<< "$endpoint_info"

    if [ "$name" = "hello" ]; then
        user_name="User$((RANDOM % 1000 + 1))"
        full_path="/api/hello/$user_name"
    else
        full_path="$path"
    fi

    payload="{\"path\": \"$full_path\", \"httpMethod\": \"$method\"}"

    # Invoke Lambda function
    response=$(aws --profile "$PROFILE" --region "$REGION" lambda invoke \
        --function-name "$FUNCTION_NAME" \
        --payload fileb://<(echo "$payload") \
        /tmp/lambda-response.json 2>&1)

    if [ $? -eq 0 ]; then
        # Parse response
        if [ -f /tmp/lambda-response.json ]; then
            status_code=$(cat /tmp/lambda-response.json | jq -r '.statusCode // 200' 2>/dev/null || echo "200")

            if [ "$status_code" = "200" ]; then
                success_count=$((success_count + 1))
                status_icon="SUCCESS"
            else
                error_count=$((error_count + 1))
                status_icon="WARNING"
            fi
        else
            error_count=$((error_count + 1))
            status_icon="ERROR"
            status_code="ERROR"
        fi

        # Print progress
        printf "%s Request %2d: %-8s | Status: %s | Path: %s\n" \
            "$status_icon" "$i" "$name" "$status_code" "$full_path"

        # Clean up
        rm -f /tmp/lambda-response.json

    else
        error_count=$((error_count + 1))
        printf "ERROR Request %2d: %-8s | Error: %s\n" "$i" "$name" "$response"
    fi

    # Add small delay between requests
    sleep $(echo "scale=2; $RANDOM/32767 * 0.4 + 0.1" | bc -l)
done

# Print summary
echo
echo "=================================================="
echo "TRAFFIC GENERATION SUMMARY"
echo "=================================================="
echo "Total Requests: $NUM_REQUESTS"
echo "Successful:     $success_count ($(echo "scale=1; $success_count * 100 / $NUM_REQUESTS" | bc -l)%)"
echo "Errors:         $error_count ($(echo "scale=1; $error_count * 100 / $NUM_REQUESTS" | bc -l)%)"
echo "Timestamp:      $(date '+%Y-%m-%d %H:%M:%S')"
echo
echo "Next Steps:"
echo "1. Wait 5-10 minutes for Application Signals to process data"
echo "2. Check Application Signals console:"
echo "   https://us-east-1.console.aws.amazon.com/cloudwatch/home?region=us-east-1#application-signals:services"
echo "3. Look for 'lambda-dotnet' in the services list"
echo

if [ $success_count -gt 0 ]; then
    echo "Traffic generation completed successfully!"
    exit 0
else
    echo "Traffic generation failed!"
    exit 1
fi

故障排除

问题：Shell 环境问题

问题： 命令因以下错误而失败： parse error near 'cursor_snap_ENV_VARS...' or command not found: dump_zsh_state

原因： Cursor IDE 环境变量注入正在破坏 shell 环境

解决方案：

  1. 使用新的终端窗口 在 Cursor IDE 之外
  2. 重启 Cursor IDE 完全关闭后重试
  3. 使用 VS Code 代替 Cursor IDE
  4. 使用 Lambda 控制台方法 （推荐在 shell 问题时使用）

问题：角色 ARN 为空

问题： aws: error: argument --role: 需要一个参数

原因： The ROLE_ARN variable is empty, indicating IAM role creation failed

调试步骤：

  1. Check if role 是否存在： 
    aws iam get-role --role-name lambda-dotnet-execution-role
  2. 如果缺失则创建角色： 
    aws iam create-role \
  --role-name lambda-dotnet-execution-role \
  --assume-role-policy-document '{
    "Version": "2012-10-17",
    "Statement": [{
      "Effect": "Allow",
      "Principal": {"Service": "lambda.amazonaws.com"},
      "Action": "sts:AssumeRole"
    }]
  }'
  3. 手动获取角色 ARN： 
    aws iam get-role \
  --role-name lambda-dotnet-execution-role \
  --query 'Role.Arn' \
  --output text
  4. 在 create-function 命令中直接使用 ARN

问题：命令语法错误

常见错误：

  • 在同一行使用句号分隔多个命令
  • 使用重复的参数 (e.g., --role twice)
  • 变量缺少引号s

❌ 错误 - 不要这样做：

# WRONG: Commands on same line with period
aws iam create-role \ ... }. aws lambda create-function \ ...

# WRONG: Duplicate --role parameter
aws lambda create-function \
  --role ${ROLE_ARN} \
  --role lambda-dotnet-execution-role

# WRONG: Missing quotes around variable
--role $ROLE_ARN

✅ 正确 - 应该这样做：

# Step 1: Create IAM role (separate command)
aws iam create-role \
  --role-name lambda-dotnet-execution-role \
  --assume-role-policy-document '{
    "Version": "2012-10-17",
    "Statement": [{
      "Effect": "Allow",
      "Principal": {"Service": "lambda.amazonaws.com"},
      "Action": "sts:AssumeRole"
    }]
  }'

# Step 2: Attach policies (separate commands)
aws iam attach-role-policy \
  --role-name lambda-dotnet-execution-role \
  --policy-arn arn:aws:iam::aws:policy/service-role/AWSLambdaBasicExecutionRole

aws iam attach-role-policy \
  --role-name lambda-dotnet-execution-role \
  --policy-arn arn:aws:iam::aws:policy/CloudWatchLambdaApplicationSignalsExecutionRolePolicy

# Step 3: Get role ARN
ROLE_ARN=$(aws iam get-role \
  --role-name lambda-dotnet-execution-role \
  --query 'Role.Arn' \
  --output text)

# Step 4: Create Lambda function (with quotes around variable)
aws lambda create-function \
  --function-name lambda-dotnet \
  --runtime dotnet9 \
  --role "$ROLE_ARN" \
  --handler lambda_dotnet::lambda_dotnet.Program::FunctionHandlerAsync \
  --zip-file fileb://bin/Release/net8.0/lambda-dotnet.zip \
  --timeout 30 \
  --memory-size 512

问题：服务未出现在 Application Signals 中

检查：

  • 验证您的账户已启用 Application Signals
  • 检查 OpenTelemetry 层是否已附加
  • 验证环境变量设置正确
  • 生成流量后等待 5-10 分钟
  • 确保 IAM 策略 CloudWatchLambdaApplicationSignalsExecutionRolePolicy 已附加

问题：Lambda 函数错误

常见问题：

  • 缺少 IAM 策略 - Ensure CloudWatchLambdaApplicationSignalsExecutionRolePolicy 已附加
  • 错误的层 ARN - 验证您所在区域的层 ARN
  • Missing 环境变量 - Ensure AWS_LAMBDA_EXEC_WRAPPER=/opt/otel-instrument
  • Handler 不匹配 - 确保 handler 为 lambda_dotnet::lambda_dotnet.Program::FunctionHandlerAsync
  • Zip 文件未找到 - 确保 dotnet 打包成功完成

Issue: 权限拒绝错误

检查您的 AWS 凭证：

  • 验证 AWS 配置文件： aws configure list
  • 检查调用者身份： aws sts get-caller-identity
  • 确保您具有所需的 IAM 权限
