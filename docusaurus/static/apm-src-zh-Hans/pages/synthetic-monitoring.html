Synthetic Monitoring（金丝雀）

Synthetics 如何与 Application Signals 连接

当您为金丝雀启用 X-Ray 追踪时，CloudWatch Synthetics 金丝雀可以出现在 Application Signals 中。这使得金丝雀调用可以与后端服务关联，并显示在服务详情和应用程序拓扑图中。

参见：Application Signals 如何实现运营健康监控。

常见金丝雀用例

  • API 健康检查：按计划验证关键端点
  • 浏览器流程：验证登录/结账旅程
  • 多区域：在客户之前检测区域性问题
