Synthetic Monitoring (Canaries)

How Synthetics connects to Application Signals

CloudWatch Synthetics canaries can appear in Application Signals when you enable X-Ray tracing for the canary. This allows canary calls to be associated with backend services and shown in service details and on the Application Map.

See: How Application Signals enables operational health monitoring.

Common canary use cases

  • API health checks: validate key endpoints on a schedule
  • Browser flows: validate login/checkout journeys
  • Multi-region: detect regional issues before customers do
