Real User Monitoring (RUM)

RUM Application Signals-உடன் எவ்வாறு இணைகிறது

RUM client-க்கு X-Ray tracing-ஐ இயக்கும்போது CloudWatch RUM web clients Application Signals-ல் தோன்றலாம். Client-லிருந்து backend சேவைகளுக்கான கோரிக்கைகள் பின்னர் தொடர்புபடுத்தப்பட்டு சேவை விவரங்கள் மற்றும் Application Map-ல் காட்டப்படும்.

பார்க்கவும்: Application Signals செயல்பாட்டு நிலை கண்காணிப்பை எவ்வாறு இயக்குகிறது.

RUM எந்தக் கேள்விகளுக்கு பதிலளிக்க உதவுகிறது

  • இந்த சிக்கல் பயனர்களை பாதிக்கிறதா? backend health-ஐ page load மற்றும் JS error signals-உடன் correlate செய்யுங்கள்
  • பயனர்கள் எங்கே பாதிக்கப்படுகிறார்கள்? புவியியல், device மற்றும் browser அடிப்படையில் செயல்திறனை ஒப்பிடுங்கள்
  • என்ன மாறியது? regressions-ஐ ஆராயும்போது deployments/change events-உடன் correlate செய்யுங்கள்
