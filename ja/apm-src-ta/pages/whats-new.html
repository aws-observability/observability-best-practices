ஜூன் 2024: Application Signals பொது கிடைக்கும் தன்மை

CloudWatch Application Signals பொதுவாக கிடைக்கும் (GA) நிலையை அடைந்தது. இது அழைப்பு எண்ணிக்கை, கிடைக்கும் தன்மை, தாமதம், தவறுகள் மற்றும் பிழைகள் போன்ற நிலையான பயன்பாட்டு மெட்ரிக்ஸை தானாக சேகரிப்பதன் மூலம் விரைவான service-level பகுப்பாய்வு மற்றும் பிழைத்திருத்தத்தில் கவனம் செலுத்துகிறது.

AWS அறிவிப்பு பாருங்கள்: GA அறிவிப்பு.

செப்டம்பர் 2024: Request-based SLOs

AWS CloudWatch Application Signals-க்கு request-based service level objectives (SLOs) ஆதரவை அறிவித்தது.

AWS அறிவிப்பு பாருங்கள்: Request-based SLOs.

தொடர்ச்சி: Application Map மற்றும் பகுப்பாய்வு பணிப்பாய்வு

CloudWatch Application Map Application Signals-க்கான "Service Map" terminology-ஐ மாற்றுகிறது. இது topology-ஐ (clients, canaries, சேவைகள், சார்புகள்) காட்சிப்படுத்துகிறது மற்றும் குழுக்கள் சம்பவங்களை பகுப்பாய்வு செய்ய உதவ SLO/SLI நிலையைக் காட்ட முடியும்.
