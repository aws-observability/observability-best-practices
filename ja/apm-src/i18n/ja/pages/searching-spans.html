スパンの検索と分析

Transaction Search

CloudWatch Transaction Search は、スパン属性を使用して取り込まれたスパンを検索・分析するためのインタラクティブな分析エクスペリエンスです。以下をサポートしています：

  • リスト：個々のトランザクションのトラブルシューティング、遅いスパンの検出、エラーの検出
  • 時系列：レイテンシーと頻度のトレンドを可視化（P50/P90/P99）
  • グループ分析：属性による集計（例：影響を受けた上位顧客）

参照： スパンの検索と分析.

スパンに対する Logs Insights

CloudWatch Logs Insights を使用してスパンをクエリすることもできます（例：DB ステートメント別の p99 レイテンシー、HTTP 5xx のスパン数）。

参照： Transaction Search.
