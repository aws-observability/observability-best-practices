Release Notes

Release notes: where to look

CloudWatch Application Signals is a managed AWS capability. Public release notes and feature announcements are published through AWS documentation and AWS “What’s New”.

  • AWS “What’s New”: aws.amazon.com/about-aws/whats-new
  • Application Signals docs: CloudWatch Application Signals

Instrumentation components

Your “versioned” upgrades typically come from the instrumentation distribution and its dependencies (for example, AWS Distro for OpenTelemetry and OpenTelemetry instrumentation libraries), plus CloudWatch Agent / platform add-ons.

  • ADOT: follow the AWS Distro for OpenTelemetry release notes appropriate for your language/runtime.
  • CloudWatch Agent: upgrade the agent when required for new Application Signals capabilities (for example, grouping via resource attributes).
