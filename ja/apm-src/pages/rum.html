Real User Monitoring (RUM)

How RUM connects to Application Signals

CloudWatch RUM web clients can appear in Application Signals when you enable X-Ray tracing for the RUM client. Requests from the client to backend services are then associated and shown in service details and on the Application Map.

See: How Application Signals enables operational health monitoring.

What RUM helps you answer

  • Is this issue user-impacting? correlate backend health to page load and JS error signals
  • Where are users impacted? compare performance by geography, device, and browser
  • What changed? correlate with deployments/change events when investigating regressions
