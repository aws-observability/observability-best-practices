Searching and analyzing spans

Transaction Search

CloudWatch Transaction Search is an interactive analytics experience for searching and analyzing ingested spans using span attributes. It supports:

  • List: troubleshoot individual transactions, find slow spans, find errors
  • Time series: visualize latency and frequency trends (P50/P90/P99)
  • Group analysis: aggregate by attributes (for example, top customers impacted)

See: Searching and analyzing spans.

Logs Insights on spans

You can also query spans using CloudWatch Logs Insights (for example, p99 duration by a DB statement, or counts of spans with HTTP 5xx).

See: Transaction Search.
