Notes de version : ou chercher

CloudWatch Application Signals est une fonctionnalite geree par AWS. Les notes de version publiques et les annonces de fonctionnalites sont publiees via la documentation AWS et la section « Quoi de neuf » d'AWS.

  • « Quoi de neuf » AWS : aws.amazon.com/about-aws/whats-new
  • Documentation Application Signals : CloudWatch Application Signals

Composants d'instrumentation

Vos mises a niveau « versionnees » proviennent generalement de la distribution d'instrumentation et de ses dependances (par exemple, AWS Distro for OpenTelemetry et les bibliotheques d'instrumentation OpenTelemetry), ainsi que de CloudWatch Agent et des extensions de plateforme.

  • ADOT : suivez les notes de version d'AWS Distro for OpenTelemetry appropriees pour votre langage/environnement d'execution.
  • CloudWatch Agent : mettez a jour l'agent lorsque cela est necessaire pour les nouvelles fonctionnalites d'Application Signals (par exemple, le regroupement via les attributs de ressource).
