Recherche et analyse de spans

Transaction Search

CloudWatch Transaction Search est une experience d'analytique interactive pour rechercher et analyser les spans ingeres a l'aide des attributs de span. Il prend en charge :

  • Liste : depanner les transactions individuelles, trouver les spans lents, trouver les erreurs
  • Series temporelles : visualiser les tendances de latence et de frequence (P50/P90/P99)
  • Analyse de groupe : agreger par attributs (par exemple, principaux clients impactes)

Voir : Recherche et analyse de spans.

Logs Insights sur les spans

Vous pouvez egalement interroger les spans a l'aide de CloudWatch Logs Insights (par exemple, la duree p99 par instruction de base de donnees, ou le nombre de spans avec HTTP 5xx).

Voir : Transaction Search.
