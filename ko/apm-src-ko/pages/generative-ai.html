생성형 AI 관찰성

Application Signals + Amazon Bedrock

CloudWatch Application Signals는 Amazon Bedrock 모델과 상호 작용하는 생성형 AI 애플리케이션의 문제 해결을 지원합니다. Bedrock API 호출에 대한 즉시 사용 가능한 텔레메트리를 제공하여 다음을 조사할 수 있습니다:

  • 모델 구성 문제
  • 사용량 및 비용 요인 (토큰 사용량)
  • 모델 지연 시간
  • 응답의 중단/완료 이유

세분화된 메트릭 및 트레이스

Bedrock 호출의 경우, Application Signals는 리소스 수준의 세부 정보(예: 모델 ID, 가드레일 ID, 지식 기반 ID, Bedrock 에이전트 ID)와 함께 성능 메트릭 및 트레이스 스팬을 생성하고 상관시킬 수 있습니다.

참조: 예제: Amazon Bedrock과 상호 작용하는 생성형 AI 앱 문제 해결.

GenAI 시맨틱 속성 + Transaction Search

Application Signals는 OpenTelemetry GenAI 시맨틱 규약 속성(예: gen_ai.request.model 및 토큰 사용량 필드)을 생성할 수 있습니다. CloudWatch Transaction Search를 사용하여 모델 간 프롬프트, 토큰 사용량, 지연 시간을 분석하고 비교할 수 있습니다.

  • Transaction Search
  • 스팬 검색 및 분석
