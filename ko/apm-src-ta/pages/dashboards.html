CloudWatch APM ஆவணங்கள்

டாஷ்போர்டுகள்

உள்ளமைக்கப்பட்ட Application Signals காட்சிகள்

Application Signals CloudWatch கன்சோலில் பயன்பாடு-மையமான பணிப்பாய்வை வழங்குகிறது:

  • சேவைகள்: சேவைகளை பட்டியலிடுங்கள், அதிக தவறு/தாமதத்தை கண்டறியுங்கள், SLO-க்கள் இருக்கும்போது SLI நிலையைப் பாருங்கள்.
  • சேவை விவரம்: செயல்பாடுகள், சார்புகள் மற்றும் தொடர்புடைய ட்ரேஸ்களை ஆராயுங்கள்.
  • Application Map: டோபாலஜியை காட்சிப்படுத்துங்கள் மற்றும் சார்புகள் முழுவதும் நிலையை பரப்புங்கள்.

பார்க்கவும்: Application Signals மூலம் செயல்பாட்டு நிலை கண்காணிப்பு.

CloudWatch டாஷ்போர்டுகள் (தனிப்பயன்)

Application Signals வெளியிடும் மெட்ரிக்ஸை (ApplicationSignals namespace) பயன்படுத்தி CloudWatch டாஷ்போர்டுகளை உருவாக்கலாம், தேவைப்படும்போது அலாரங்கள்/லாக்களுடன் தொடர்புபடுத்தலாம்.

  • சேவை மெட்ரிக்ஸ்: Service/Operation/Environment பரிமாணங்கள் முழுவதும் தாமதம், தவறுகள், பிழைகள்.
  • சார்பு மெட்ரிக்ஸ்: கீழ்நிலை அழைப்புகளுக்கான remote service/operation பரிமாணங்கள்.
  • Runtime மெட்ரிக்ஸ் (ஆதரிக்கப்படும் இடங்களில்): தாமத உச்சங்களுடன் GC/CPU/நினைவகத்தை தொடர்புபடுத்துங்கள்.

பார்க்கவும்: Application Signals மூலம் சேகரிக்கப்படும் மெட்ரிக்ஸ்.
