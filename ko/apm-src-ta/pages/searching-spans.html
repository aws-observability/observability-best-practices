CloudWatch APM ஆவணங்கள்

Spans-ஐ தேடுதல் மற்றும் பகுப்பாய்வு

Transaction Search

CloudWatch Transaction Search என்பது span attributes பயன்படுத்தி ingested spans-ஐ தேடவும் பகுப்பாய்வு செய்யவும் ஒரு interactive analytics அனுபவமாகும். இது ஆதரிக்கிறது:

  • பட்டியல்: தனிப்பட்ட transactions-ஐ பிழைத்திருத்துதல், மெதுவான spans-ஐ கண்டறிதல், errors-ஐ கண்டறிதல்
  • நேரத் தொடர்: தாமதம் மற்றும் frequency trends-ஐ (P50/P90/P99) காட்சிப்படுத்துதல்
  • குழு பகுப்பாய்வு: attributes அடிப்படையில் aggregate செய்தல் (எடுத்துக்காட்டாக, பாதிக்கப்பட்ட முதன்மை வாடிக்கையாளர்கள்)

பார்க்கவும்: Spans-ஐ தேடுதல் மற்றும் பகுப்பாய்வு.

Spans மீது Logs Insights

CloudWatch Logs Insights பயன்படுத்தியும் spans-ஐ query செய்யலாம் (எடுத்துக்காட்டாக, DB statement அடிப்படையில் p99 duration, அல்லது HTTP 5xx-உடன் spans-ன் counts).
