Synthetic Monitoring (Canaries)

Synthetics が Application Signals に接続する仕組み

CloudWatch Synthetics カナリアは、カナリアの X-Ray トレースを有効にすると Application Signals に表示されます。これにより、カナリアの呼び出しをバックエンドサービスに関連付け、サービスの詳細および Application Map に表示することができます。

参照: Application Signals が運用上の健全性監視を有効にする仕組み.

カナリアの一般的なユースケース

  • API ヘルスチェック: スケジュールに従って主要なエンドポイントを検証する
  • ブラウザフロー: ログイン／チェックアウトのジャーニーを検証する
  • マルチリージョン: 顧客より先にリージョンの問題を検出する
