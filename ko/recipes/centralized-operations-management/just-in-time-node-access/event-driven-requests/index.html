JITNA와 EventBridge 통합

Just-in-time 노드 접근(JITNA)을 사용할 때, 수동 승인이 필요한 세션 접근 요청이 생성되면 EventBridge로 세 가지 유형의 이벤트가 전송됩니다:

  1. 요청자 접근 요청 상태 업데이트
  2. 승인자 접근 요청 상태 업데이트
  3. JITNA 접근 요청 실패
정보

자동 승인 정책에 따라 자동으로 승인된 접근 요청에 대해서는 이벤트가 전송되지 않습니다.
