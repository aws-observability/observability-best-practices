FirehoseмҷҖ AWS LambdaлҘј нҶөн•ҙ CloudWatch Metric StreamsлҘј Amazon Managed Service for PrometheusлЎң лӮҙліҙлӮҙкё°

мқҙ л ҲмӢңн”јм—җм„ңлҠ” CloudWatch Metric Streamмқ„ кі„мёЎн•ҳкі  Kinesis Data FirehoseмҷҖ AWS LambdaлҘј мӮ¬мҡ©н•ҳм—¬ Amazon Managed Service for Prometheus (AMP)лЎң л©”нҠёлҰӯмқ„ мҲҳм§‘н•ҳлҠ” л°©лІ•мқ„ ліҙм—¬мӨҚлӢҲлӢӨ.

мҷ„м „н•ң мӢңлӮҳлҰ¬мҳӨлҘј мӢңм—°н•ҳкё° мң„н•ҙ AWS Cloud Development Kit (CDK)лҘј мӮ¬мҡ©н•ҳм—¬ Firehose Delivery Stream, Lambda, S3 Bucketмқ„ мғқм„ұн•ҳлҠ” мҠӨнғқмқ„ м„Өм •н•©лӢҲлӢӨ.

л…ёнҠё

мқҙ к°Җмқҙл“ңлҘј мҷ„лЈҢн•ҳлҠ” лҚ° м•Ҫ 30л¶„мқҙ мҶҢмҡ”лҗ©лӢҲлӢӨ.

мқён”„лқјвҖӢ

лӢӨмқҢ м„№м…ҳм—җм„ңлҠ” мқҙ л ҲмӢңн”јлҘј мң„н•ң мқён”„лқјлҘј м„Өм •н•©лӢҲлӢӨ.

CloudWatch Metric StreamsлҘј мӮ¬мҡ©н•ҳл©ҙ мҠӨнҠёлҰ¬л°Қ л©”нҠёлҰӯ лҚ°мқҙн„°лҘј HTTP м—”л“ңнҸ¬мқёнҠё лҳҗлҠ” S3 лІ„нӮ·мңјлЎң м „лӢ¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мӮ¬м „ мҡ”кө¬ мӮ¬н•ӯвҖӢ

  • AWS CLIк°Җ нҷҳкІҪм—җ м„Өм№ҳ л°Ҹ кө¬м„ұлҗҳм–ҙ мһҲм–ҙм•ј н•©лӢҲлӢӨ.
  • AWS CDK Typescriptк°Җ нҷҳкІҪм—җ м„Өм№ҳлҗҳм–ҙ мһҲм–ҙм•ј н•©лӢҲлӢӨ.
  • Node.js л°Ҹ Goк°Җ м„Өм№ҳлҗҳм–ҙ мһҲм–ҙм•ј н•©лӢҲлӢӨ.
  • м ҖмһҘмҶҢк°Җ лЎңм»¬ лЁёмӢ м—җ нҒҙлЎ лҗҳм–ҙ мһҲм–ҙм•ј н•©лӢҲлӢӨ. мқҙ н”„лЎңм қнҠёмқҳ мҪ”л“ңлҠ” /sandbox/CWMetricStreamExporter м•„лһҳм—җ мһҲмҠөлӢҲлӢӨ.

AMP мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨ мғқм„ұвҖӢ

мқҙ л ҲмӢңн”јмқҳ лҚ°лӘЁ м• н”ҢлҰ¬мјҖмқҙм…ҳмқҖ AMP мң„м—җм„ң мӢӨн–үлҗ©лӢҲлӢӨ. лӢӨмқҢ лӘ…л №мңјлЎң AMP мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨлҘј мғқм„ұн•©лӢҲлӢӨ:

лӢӨмқҢ лӘ…л №мңјлЎң мӣҢнҒ¬мҠӨнҺҳмқҙмҠӨк°Җ мғқм„ұлҗҳм—ҲлҠ”м§Җ нҷ•мқён•©лӢҲлӢӨ:

aws amp list-workspaces

лӢӨмқҢ лӘ…л №мңјлЎң /cdk л””л үнҶ лҰ¬лЎң мқҙлҸҷн•©лӢҲлӢӨ:
