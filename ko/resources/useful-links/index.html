유용한 링크

AWS Observability 모범 사례를 학습하고 구현하는 데 도움이 되는 엄선된 리소스 모음입니다.

예정된 이벤트 등록 (AWS Connected Community)​

개발자, DevOps 엔지니어, 솔루션 아키텍트, 시스템 관리자 및 클라우드 운영에 관심 있는 모든 분을 위한 라이브 웨비나 및 워크숍에 여기에서 등록하세요

AWS re:Invent 2025 Cloud Operations (COP) 세션 (YouTube 재생목록)​

여기에서 Cloud Operations (COP) 세션의 AWS re:Invent 2025 브레이크아웃 세션 재생목록을 확인하세요

대화형 데모​

AWS Cloud Operations Show​

흥미로운 AWS Cloud Operations 기능 및 사용 사례를 심층적으로 다루는 라이브 스트리밍 녹화 모음입니다. 이 비디오 시리즈는 클라우드 운영에 대한 실용적인 데모와 전문가 인사이트를 제공합니다.

YouTube에서 보기

One Observability Workshop​

AWS가 모니터링 및 Observability를 위해 제공하는 다양한 도구 세트에 대한 실습 경험을 제공하는 핸즈온 워크숍입니다. 이 워크숍은 관심사와 가용 시간에 따라 모듈을 선택할 수 있어 높은 맞춤화가 가능합니다.

AWS Distro for OpenTelemetry (ADOT)​

ADOT로 코드를 계측하는 방법에 대한 종합 가이드입니다. AWS가 지원하는 OpenTelemetry 배포판을 사용하여 애플리케이션에서 분산 트레이스 및 메트릭을 수집하는 방법을 알아보세요.
