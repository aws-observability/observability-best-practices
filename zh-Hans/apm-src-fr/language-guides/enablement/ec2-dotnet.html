Overview

This guide provides step-by-step instructions to deploy a .NET Core application on Amazon EC2 with AWS Application Signals enabled. This is the EC2 equivalent of the Lambda .NET function, reusing the same business logic and endpoints.

What you'll accomplish:
  • Deploy a .NET Core app on EC2
  • Enable AWS Application Signals auto-instrumentation
  • View your service in CloudWatch Application Signals console
  • Monitor service metrics, traces, and operations
Time to complete: 30-45 minutes

Prerequisites

Required Tools

  • AWS CLI v2 (latest version)
  • Docker Desktop
  • .NET 8.0 SDK (for local development)
  • AWS Account with appropriate permissions
  • Text editor

Important Compatibility Requirements

  • AWS SDK .NET V3: The AWS Distro for OpenTelemetry (ADOT) SDK for .NET does not support AWS SDK for .NET V4. Use AWS SDK .NET V3 for full Application Signals support.
  • OpenTelemetry Packages: Use compatible OpenTelemetry packages with AWS.DistributedOpenTelemetry for Application Signals integration.

AWS Permissions Required

  • EC2: Full access (create instances, security groups, key pairs)
  • IAM: Create roles and policies
  • CloudWatch: Logs and Application Signals access
  • VPC: Create and manage VPC resources
  • SSM: Parameter Store write access

Knowledge Assumed

  • Basic AWS CLI usage
  • Basic Docker commands
  • Basic understanding of EC2 concepts
  • Basic .NET Core knowledge

Architecture

Your EC2 instance will run:

  1. .NET Core Application: Your web API with Application Signals instrumentation
  2. CloudWatch Agent: Collects traces and metrics (optional, can use direct OTLP)

Flow:

EC2 Instance
  → .NET Core App (auto-instrumented)
  → OpenTelemetry Collector (optional)
  → CloudWatch Application Signals

1.3 Create the Application Code

Create Program.cs with the same endpoints as the Lambda function:

using Microsoft.AspNetCore.Builder;
using Microsoft.AspNetCore.Hosting;
using Microsoft.AspNetCore.Http;
using Microsoft.Extensions.DependencyInjection;
using Microsoft.Extensions.Hosting;
using Microsoft.Extensions.Logging;
using System;
using System.Collections.Generic;
using System.Text.Json;
using System.Threading.Tasks;
using System.Threading;
using System.Net.Http;
using Amazon.S3;
using Amazon.S3.Model;

namespace ec2_dotnet
{
    public class Program
    {
        public static void Main(string[] args)
        {
            CreateHostBuilder(args).Build().Run();
        }

        public static IHostBuilder CreateHostBuilder(string[] args) =>
            Host.CreateDefaultBuilder(args)
                .ConfigureWebHostDefaults(webBuilder =>
                {
                    webBuilder.UseStartup<Startup>();
                });
    }

    public class Startup
    {
        public void ConfigureServices(IServiceCollection services)
        {
            services.AddControllers();
            services.AddHttpClient();
            services.AddAWSService<IAmazonS3>();
        }

        public void Configure(IApplicationBuilder app, IWebHostEnvironment env)
        {
            if (env.IsDevelopment())
            {
                app.UseDeveloperExceptionPage();
            }

            app.UseRouting();
            app.UseCors(builder => builder
                .AllowAnyOrigin()
                .AllowAnyMethod()
                .AllowAnyHeader());

            app.UseEndpoints(endpoints =>
            {
                endpoints.MapControllers();
            });
        }
    }

    [Microsoft.AspNetCore.Mvc.ApiController]
    [Microsoft.AspNetCore.Mvc.Route("")]
    public class DemoController : Microsoft.AspNetCore.Mvc.ControllerBase
    {
        private readonly ILogger<DemoController> _logger;
        private readonly HttpClient _httpClient;
        private readonly IAmazonS3 _s3Client;
        private readonly Random _random = new Random();

        public DemoController(ILogger<DemoController> logger, HttpClient httpClient, IAmazonS3 s3Client)
        {
            _logger = logger;
            _httpClient = httpClient;
            _s3Client = s3Client;
        }

        [Microsoft.AspNetCore.Mvc.HttpGet]
        public IActionResult Root()
        {
            return Ok(new
            {
                message = "Hello World from .NET EC2!",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet",
                runtime = "dotnet8",
                host = Environment.MachineName,
                environment = "ec2"
            });
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("health")]
        public IActionResult Health()
        {
            return Ok(new
            {
                status = "healthy",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet",
                runtime = "dotnet8",
                host = Environment.MachineName
            });
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/hello/{name}")]
        public async Task<IActionResult> Hello(string name)
        {
            // Simulate some processing time
            await Task.Delay(_random.Next(10, 110));

            return Ok(new
            {
                message = $"Hello, {name}!",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet",
                runtime = "dotnet8",
                greeting_id = _random.Next(10000)
            });
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/slow")]
        public async Task<IActionResult> Slow()
        {
            // Simulate slow operation (500-1500ms)
            var delay = _random.Next(500, 1500);
            await Task.Delay(delay);

            return Ok(new
            {
                message = "Slow operation completed",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet",
                duration_ms = delay
            });
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/error")]
        public IActionResult Error()
        {
            if (_random.NextDouble() > 0.5)
            {
                throw new Exception("Simulated error for Application Signals testing");
            }

            return Ok(new
            {
                message = "No error occurred",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet"
            });
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/external")]
        public async Task<IActionResult> External()
        {
            try
            {
                var response = await _httpClient.GetAsync("https://httpbin.org/delay/1");

                return Ok(new
                {
                    message = "External call completed",
                    external_status_code = (int)response.StatusCode,
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    service = "ec2-dotnet"
                });
            }
            catch (Exception ex)
            {
                throw new Exception($"External call failed: {ex.Message}");
            }
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/s3")]
        public async Task<IActionResult> S3()
        {
            try
            {
                var response = await _s3Client.ListBucketsAsync();

                return Ok(new
                {
                    message = "S3 call completed successfully",
                    bucket_count = response.Buckets.Count,
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    service = "ec2-dotnet",
                    first_bucket = response.Buckets.Count > 0 ? response.Buckets[0].BucketName : null
                });
            }
            catch (Exception ex)
            {
                return StatusCode(500, new
                {
                    error = "S3 call failed",
                    message = ex.Message,
                    timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                    service = "ec2-dotnet"
                });
            }
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/database")]
        public async Task<IActionResult> Database()
        {
            // Simulate database query time
            await Task.Delay(_random.Next(50, 250));

            return Ok(new
            {
                message = "Database query completed",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet",
                records_returned = _random.Next(1, 101)
            });
        }

        [Microsoft.AspNetCore.Mvc.HttpGet("api/complex")]
        public async Task<IActionResult> Complex()
        {
            // Step 1: Simulate authentication
            await Task.Delay(_random.Next(10, 60));

            // Step 2: Simulate business logic
            await Task.Delay(_random.Next(20, 120));

            // Step 3: Simulate database operations
            await Task.Delay(_random.Next(30, 180));

            return Ok(new
            {
                message = "Complex operation completed successfully",
                timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
                service = "ec2-dotnet",
                steps_completed = 3,
                total_processing_time_ms = _random.Next(60, 360)
            });
        }
    }

1.4 Create Dockerfile

Create Dockerfile:

# Use the official .NET 8 runtime as base image
FROM mcr.microsoft.com/dotnet/aspnet:8.0 AS base
WORKDIR /app
EXPOSE 8080

# Use the .NET 8 SDK for building
FROM mcr.microsoft.com/dotnet/sdk:8.0 AS build
WORKDIR /src
COPY ["ec2-dotnet.csproj", "."]
RUN dotnet restore "ec2-dotnet.csproj"
COPY . .
RUN dotnet build "ec2-dotnet.csproj" -c Release -o /app/build

FROM build AS publish
RUN dotnet publish "ec2-dotnet.csproj" -c Release -o /app/publish

FROM base AS final
WORKDIR /app
COPY --from=publish /app/publish .

# Set environment variables for Application Signals
ENV OTEL_SERVICE_NAME=ec2-dotnet
ENV OTEL_RESOURCE_ATTRIBUTES=service.name=ec2-dotnet,service.namespace=aws-observability,service.version=1.0.0,deployment.environment=ec2
ENV OTEL_EXPORTER_OTLP_ENDPOINT=https://otlp.nr-data.net
ENV OTEL_EXPORTER_OTLP_HEADERS=api-key=your-api-key
ENV OTEL_TRACES_EXPORTER=otlp
ENV OTEL_METRICS_EXPORTER=otlp
ENV OTEL_LOGS_EXPORTER=otlp

ENTRYPOINT ["dotnet", "ec2-dotnet.dll"]

Step 3: Deploy to EC2

3.1 Create EC2 Instance

# Create a key pair
aws ec2 create-key-pair --key-name ec2-dotnet-key --query 'KeyMaterial' --output text > ec2-dotnet-key.pem
chmod 400 ec2-dotnet-key.pem

# Create security group
aws ec2 create-security-group \
    --group-name ec2-dotnet-sg \
    --description "Security group for .NET EC2 Application Signals demo"

# Get security group ID
SG_ID=$(aws ec2 describe-security-groups --group-names ec2-dotnet-sg --query 'SecurityGroups[0].GroupId' --output text)

# Allow HTTP traffic
aws ec2 authorize-security-group-ingress \
    --group-id $SG_ID \
    --protocol tcp \
    --port 8080 \
    --cidr 0.0.0.0/0

# Allow SSH
aws ec2 authorize-security-group-ingress \
    --group-id $SG_ID \
    --protocol tcp \
    --port 22 \
    --cidr 0.0.0.0/0

# Launch EC2 instance
aws ec2 run-instances \
    --image-id ami-0c02fb55956c7d316 \
    --count 1 \
    --instance-type t3.medium \
    --key-name ec2-dotnet-key \
    --security-group-ids $SG_ID \
    --user-data '#!/bin/bash
yum update -y
yum install -y docker
systemctl start docker
systemctl enable docker
usermod -a -G docker ec2-user
yum install -y git
curl -sSL https://dot.net/v1/dotnet-install.sh | bash /dev/stdin --channel 8.0
echo "export PATH=$PATH:/home/ec2-user/.dotnet" >> /home/ec2-user/.bashrc'

3.2 Get Instance Information

# Get instance ID
INSTANCE_ID=$(aws ec2 describe-instances --filters "Name=instance-state-name,Values=running" "Name=key-name,Values=ec2-dotnet-key" --query 'Reservations[0].Instances[0].InstanceId' --output text)

# Get public IP
PUBLIC_IP=$(aws ec2 describe-instances --instance-ids $INSTANCE_ID --query 'Reservations[0].Instances[0].PublicIpAddress' --output text)

echo "Instance ID: $INSTANCE_ID"
echo "Public IP: $PUBLIC_IP"

3.3 Deploy Application

# Copy files to EC2
scp -i ec2-dotnet-key.pem -r . ec2-user@$PUBLIC_IP:/home/ec2-user/ec2-dotnet/

# SSH into instance
ssh -i ec2-dotnet-key.pem ec2-user@$PUBLIC_IP

# On the EC2 instance, navigate to the project
cd ec2-dotnet

# Install .NET 8
curl -sSL https://dot.net/v1/dotnet-install.sh | bash /dev/stdin --channel 8.0
export PATH=$PATH:/home/ec2-user/.dotnet

# Build the application
dotnet build

# Run the application
./start.sh

Step 5: Generate Traffic and Monitor

5.1 Generate Traffic

Create generate-traffic.sh:

#!/bin/bash

BASE_URL="http://$PUBLIC_IP:8080"

echo "Generating traffic for .NET EC2 Application Signals demo..."

# Generate traffic for 5 minutes
for i in {1..300}; do
    echo "Request $i"

    # Root endpoint
    curl -s "$BASE_URL" > /dev/null

    # Health check
    curl -s "$BASE_URL/health" > /dev/null

    # Hello endpoint
    curl -s "$BASE_URL/api/hello/User$i" > /dev/null

    # Slow endpoint
    curl -s "$BASE_URL/api/slow" > /dev/null

    # Error endpoint
    curl -s "$BASE_URL/api/error" > /dev/null

    # External call
    curl -s "$BASE_URL/api/external" > /dev/null

    # S3 call
    curl -s "$BASE_URL/api/s3" > /dev/null

    # Database simulation
    curl -s "$BASE_URL/api/database" > /dev/null

    # Complex operation

Troubleshooting

Common Issues

  1. Application won't start

    • Check .NET 8 is installed: dotnet --version
    • Verify port 8080 is open in security group
    • Check application logs

  2. No data in Application Signals

    • Verify environment variables are set
    • Check CloudWatch agent is running
    • Ensure IAM role has proper permissions

  3. S3 calls failing

    • Attach IAM role with S3 read permissions
    • Verify AWS credentials are configured
