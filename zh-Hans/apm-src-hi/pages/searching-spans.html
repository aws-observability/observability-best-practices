CloudWatch APM दस्तावेज़ीकरण

स्पैन खोजना और विश्लेषण करना

Transaction Search

CloudWatch Transaction Search स्पैन एट्रिब्यूट का उपयोग करके इनजेस्ट किए गए स्पैन को खोजने और विश्लेषण करने के लिए एक इंटरैक्टिव एनालिटिक्स अनुभव है। यह समर्थन करता है:

  • लिस्ट: व्यक्तिगत ट्रांज़ैक्शन की समस्या निवारण, धीमे स्पैन खोजें, एरर खोजें
  • टाइम सीरीज़: लेटेंसी और फ़्रीक्वेंसी ट्रेंड (P50/P90/P99) विज़ुअलाइज़ करें
  • ग्रुप एनालिसिस: एट्रिब्यूट द्वारा एग्रीगेट करें (उदाहरण के लिए, शीर्ष प्रभावित ग्राहक)

देखें: स्पैन खोजना और विश्लेषण करना।

स्पैन पर Logs Insights

आप CloudWatch Logs Insights का उपयोग करके भी स्पैन क्वेरी कर सकते हैं (उदाहरण के लिए, DB स्टेटमेंट द्वारा p99 ड्यूरेशन, या HTTP 5xx वाले स्पैन की गिनती)।

देखें: Transaction Search।
