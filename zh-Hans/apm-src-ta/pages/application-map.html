CloudWatch APM ஆவணங்கள்

Application Map

இது என்ன

CloudWatch Application Map உங்கள் பயன்பாட்டு topology மற்றும் செயல்பாட்டு ஆரோக்கியத்தை காட்சிப்படுத்துகிறது. இது Application Signals-க்கான பழைய "Service Map" சொல்லாட்சியை மாற்றுகிறது.

இது clients, canaries, சேவைகள் மற்றும் சார்புகளுக்கு இடையேயான இணைப்புகளைக் காட்டுகிறது, குழுக்கள் செயல்படுத்தல் ஓட்டத்தைப் புரிந்துகொள்ளவும், தவறுகள் மற்றும் தாமதம் எங்கிருந்து உருவாகின்றன என்பதை விரைவாகக் கண்டறியவும் உதவுகிறது.

முக்கிய திறன்கள்

  • SLO/SLI ஆரோக்கியம்: எந்த சேவைகள் இலக்குகளை பூர்த்தி செய்கின்றன அல்லது மீறுகின்றன, பாதிப்பு எவ்வாறு கீழ்நிலையில் பரவுகிறது என்பதைப் பாருங்கள்.
  • ஆழ்நிலை ஆய்வு: metrics-ஐ பார்க்கவும் சேவை விவரங்களுக்கு செல்லவும் nodes/edges-ஐ தேர்ந்தெடுக்கவும்.
  • குழுவாக்கம் மற்றும் வடிகட்டல்: இயல்புநிலை குழுவாக்கங்களை (தொடர்புடைய சேவைகள், சூழல்) பயன்படுத்தவும் அல்லது tags அல்லது OTEL resource attributes பயன்படுத்தி தனிப்பயன் குழுக்களை வரையறுக்கவும்.
  • Instrument செய்யப்படாத சேவைகள்: Application Signals இன்னும் instrument செய்யப்படாத சேவைகளை காட்டி, rollout-ஐ முன்னுரிமைப்படுத்த உதவும்.
  • குறுக்கு-கணக்கு தெரிவுநிலை: குறுக்கு-கணக்கு observability கட்டமைக்கப்பட்டிருக்கும்போது பல கணக்குகள் முழுவதும் சேவைகளை காட்சிப்படுத்தவும்.

அதிகாரப்பூர்வ ஆவணங்கள்

CloudWatch application map மூலம் உங்கள் பயன்பாட்டு topology-ஐ பார்க்கவும் செயல்பாட்டு ஆரோக்கியத்தை கண்காணிக்கவும்
