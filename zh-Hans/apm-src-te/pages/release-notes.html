విడుదల గమనికలు

విడుదల గమనికలు: ఎక్కడ చూడాలి

CloudWatch Application Signals అనేది నిర్వహించబడే AWS సామర్థ్యం. పబ్లిక్ విడుదల గమనికలు మరియు ఫీచర్ ప్రకటనలు AWS డాక్యుమెంటేషన్ మరియు AWS "What's New" ద్వారా ప్రచురించబడతాయి.

  • AWS "What's New": aws.amazon.com/about-aws/whats-new
  • Application Signals డాక్స్: CloudWatch Application Signals

ఇన్‌స్ట్రుమెంటేషన్ భాగాలు

మీ "వెర్షన్డ్" అప్‌గ్రేడ్‌లు సాధారణంగా ఇన్‌స్ట్రుమెంటేషన్ డిస్ట్రిబ్యూషన్ మరియు దాని డిపెండెన్సీల నుండి వస్తాయి (ఉదాహరణకు, AWS Distro for OpenTelemetry మరియు OpenTelemetry ఇన్‌స్ట్రుమెంటేషన్ లైబ్రరీలు), అలాగే CloudWatch Agent / ప్లాట్‌ఫారమ్ యాడ్-ఆన్‌లు.

  • ADOT: మీ భాష/రన్‌టైమ్‌కు తగిన AWS Distro for OpenTelemetry విడుదల గమనికలను అనుసరించండి.
  • CloudWatch Agent: కొత్త Application Signals సామర్థ్యాలకు అవసరమైనప్పుడు ఏజెంట్‌ను అప్‌గ్రేడ్ చేయండి (ఉదాహరణకు, రిసోర్స్ అట్రిబ్యూట్‌ల ద్వారా గ్రూపింగ్).

తెలిసిన సమస్యలు మరియు అనుకూలత

ప్రస్తుత అనుకూలత మ్యాట్రిక్స్ మరియు తెలిసిన సమస్యల కోసం, అధికారిక "మద్దతు ఉన్న సిస్టమ్‌లు" పేజీని చూడండి.
