CloudWatch APM 文档

自动检测

"自动检测"的含义

自动检测在运行时添加 OpenTelemetry 检测（无需更改应用程序代码），使服务发出 Application Signals 用于发现和健康监控的遥测数据。

常见 AWS 设置组件

  • AWS Distro for OpenTelemetry (ADOT) SDK 和/或收集器
  • CloudWatch Agent 在支持的设置中收集/导出遥测数据
  • 平台启用（EKS 插件 / ECS 边车 / EC2 代理设置 / Lambda 层）

从这里开始：Application Signals 概述。

支持的语言和系统

Application Signals 支持 Java、Python、Node.js 和 .NET 应用程序，并在 Amazon EKS、Kubernetes、Amazon ECS 和 Amazon EC2 上支持/测试。

参见：支持的系统。

注意事项和常见问题

  • Node.js ESM：ESM 支持有限；除非您遵循文档中的 ESM 特定指南，否则建议使用 CommonJS。
  • 容器中的 Python：您可能需要配置 PYTHONPATH 以用于 OTel 自动检测。
