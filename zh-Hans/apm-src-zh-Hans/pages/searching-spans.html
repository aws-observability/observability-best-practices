搜索和分析 span

Transaction Search

CloudWatch Transaction Search 是一种交互式分析体验，用于使用 span 属性搜索和分析已摄取的 span。它支持：

  • 列表：排查单个事务、查找慢速 span、查找错误
  • 时间序列：可视化延迟和频率趋势（P50/P90/P99）
  • 分组分析：按属性聚合（例如，受影响的主要客户）

参见：搜索和分析 span。

使用 Logs Insights 查询 span

您还可以使用 CloudWatch Logs Insights 查询 span（例如，按 DB 语句的 p99 持续时间，或 HTTP 5xx span 的计数）。

参见：Transaction Search。
